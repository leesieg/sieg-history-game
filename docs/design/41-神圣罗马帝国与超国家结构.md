# 41-神圣罗马帝国与超国家结构

> 承接 `36`（国家）、`38`（外交军事）、`40`（信仰，宗改→30YW→威斯特法利亚弧）后的设计讨论结论。
> 神罗当前被建模为**一个整国**，与上述「诸侯改宗争自治 → 宗教联盟 → 三十年战争 → 威斯特法利亚诸侯得主权」的历史弧**结构性矛盾**。本文设计一个 **EU4-lite 神罗系统**，并抽出一层**可复用的「超国家结构」原语**（教皇/哈里发是它的宗教版亲戚）。
> 日期：2026-06-27

---

# 第一部分 · 设计总述

## A. 现状的病

| 现状 | 病 |
|---|---|
| 神罗 = 一个 `empire` 整国 + 一个 `princes` 阶层 | doc 40 的弧线要求「一堆半主权诸侯」——**整国演不出诸侯改宗/争主权** |
| 诸侯只是国内阶层 | 无法扮演、无法各自改宗、无法结成天主教联盟 vs 新教同盟 |
| 教皇/哈里发（doc 40）是另一套跨国权威 | 与神罗的「头+成员+权威」结构其实**同构**，却各写各的 |

## B. 核心思想

> 神罗是游戏论题的**纯样本**：「封建拼图怎么变成近代主权国家?」它给两条可玩岔路——**皇帝集权成统一帝国（哈布斯堡未竟之梦）** vs **诸侯各得主权（威斯特法利亚史实）**。这条岔路用一个整国演不出来，故配专门系统。

- **超国家结构原语**：头(head) + 成员(members) + 权威(authority) + 选任 + 结构动作 + 集权↔分裂动态。
- **神罗 = 该原语的「领土邦联」旗舰实例**；**教皇/哈里发 = 「宗教权威」实例**（retro-fit doc 40）。
- **集权↔分裂 = doc 36 王权动态搬到邦联尺度**：皇帝想把神罗捏成统一绝对君主国，诸侯想散。

## C. 关键设计决策（讨论结论，已定）

- **C1. 选项 3：EU4-lite 邦联**（成员各为半主权国家 + 帝国元层），非整国、也非「整国+凝聚度」内部模型。
- **C2. ~20 成员**：7 大选帝侯 + 奥地利 + 次要邦 + 已存在采邑。**不是 EU4 的 300 小邦**——封顶 ~20。
- **C3. 诸侯国可扮演**：玩家可选勃兰登堡/萨克森/巴伐利亚等成员开局。
- **C4. 可复用「超国家结构」原语**：神罗为领土旗舰，教皇/哈里发为宗教实例，未来可装汉萨/卡尔马。

## D. 接缝 + 回到核心循环

> 帝国权威（压力层，受成员改宗/抵抗/外援驱动）→ 皇帝集权 vs 诸侯争自治（关口扰动）→ 宗教联盟 → 30YW（局势）→ 威斯特法利亚（因果链）→ 近代主权国家 → 循环。

神罗是**信仰弧（doc 40）的可玩引擎**：弧线的每一步都发生在神罗成员之间。

# 第二部分 · 落地蓝图

## 1. 可复用「超国家结构」原语

```js
// world.structures[key]
Structure = {
  key:        "hre",
  type:       "territorial",          // territorial(邦联) | religious(宗教权威) | league(联盟)
  head:       "奥地利",                // 皇帝 / 教皇 / 哈里发 / 盟主
  authority:  50,                     // 0-100：帝国/教皇/哈里发权威
  members:    [/* polity ids */],     // 成员 / 信徒 / 盟员
  selectors:  [/* 7 选帝侯 */],        // 选帝侯 / 枢机团 / —
  selection:  "election",             // election | conclave | controller | hereditary
  blocs:      { a:[], b:[] },         // 内部集团（宗教联盟/派系）
};
```
| 角色 | 领土实例（神罗） | 宗教实例（教皇/哈里发，doc 40） |
|---|---|---|
| head | 皇帝 | 教皇 / 哈里发 |
| authority | 帝国权威 | 教皇/哈里发权威 |
| members | 成员邦 | 信徒国 |
| selection | 选帝侯选举 | 枢机选举 / 控制权竞争 |
| 动作 | 帝国军/帝国和平/除籍 | 绝罚/十字军号召/捍卫者 |
| 集权↔分裂 | 统一帝国 ↔ 主权诸侯 | 教权至上 ↔ 国家教会 |

> doc 40 的教皇/哈里发**回填**为本原语的 religious 实例——两套权威共用一份代码。

## 3. 皇帝与选举

- **7 选帝侯**（#2–8）在帝位空缺（皇帝去世）时选举，复用 doc 36 继承/选举机制搬到帝国层。
- **选举影响因子**：候选实力、**贿选**（历史上巨额，花金钱拉票）、**教皇加冕/虔诚**（doc 40，教皇为皇帝加冕）、**选侯信仰**（新教选侯不投狂热天主教候选）。
- 皇帝通常是最强成员（哈布斯堡崛起后近乎垄断）；皇帝得领导权 + 帝国权威加成，但对成员只有有限直接控制。

## 4. 帝国权威（集权↔分裂 meter）

```js
structures.hre.authority  // 0-100
// ↑：皇帝个人实力、帝国改革(Reichsreform)、赢得帝国战争、信仰统一
// ↓：成员改宗(宗改)、诸侯抵抗、败战、外国干涉
```
| 权威 | 效果 |
|---|---|
| 高 (>70) | 皇帝可征**帝国军**、强制**帝国和平**（成员间禁战）、压成员内乱、强推宗教统一 → 趋向统一帝国 |
| 中 | 松散邦联，皇帝领导但不能强制 |
| 低 (<30) | 诸侯自由、各自为政、神罗名存实亡 |

> 这是 doc 36「集权代价 / 王权守恒」在邦联尺度的镜像：皇帝集权挤压诸侯自治，诸侯不满则抵抗。

## 5. 成员关系（多边帝国附庸，复用 doc 38）

- 成员各有自己的政体/经济/军队（doc 36/37/38），但作为成员：
  - **义务**（权威高时）：欠皇帝帝国赋税/兵员、守帝国和平（不可互攻）。
  - **收益**：集体防御（外敌攻成员 → 帝国军可援）、皇帝调解、帝国法律保护。
- **帝国除籍/帝国执行**：皇帝可宣告某成员为帝国公敌 → 其他成员获讨伐 CB（doc 38）。

## 6. 帝国动作（皇帝视角）

| 动作 | 效果 |
|---|---|
| 召集帝国议会 Reichstag | 推帝国改革（↑权威）或对诸侯让步（换支持） |
| 征帝国军 | 权威高时集结成员兵力 |
| 帝国除籍 | 宣成员为公敌（§5） |
| 调解成员争端 | 平内乱、攒威望 |
| 推帝国改革 vs 诸侯抵抗 | 集权拉锯（裁断式，复用 P1-2） |

## 7. 诸侯视角（玩家扮成员）

玩家选一个成员（如勃兰登堡/萨克森）开局，从**下方**体验集权↔分裂张力：
- **争自治**：降低皇帝对你的约束、抵抗帝国改革。
- **改宗**（宗改期，doc 40 §6.2）：夺教产红利 + 摆脱皇帝/教皇 → 自治，但招天主教联盟讨伐。
- **扩张**：吞并弱邻成员（权威低时帝国和平失效）。
- **竞选皇帝**：若为选侯/强邦，争帝位。
- **熬过 30YW**：在两大联盟夹缝中站队/求存。
> 勃兰登堡→普鲁士、萨克森→新教领袖，都是天然剧本。

## 8. 历史弧引擎（神罗 = doc 40 弧线的舞台）

| 阶段 | 神罗机制 |
|---|---|
| **宗改** | 成员逐个弹「改宗抉择」（doc 40 §6.2）→ **天主教联盟（皇帝侧）vs 新教同盟**（doc 38 集团）；**每次改宗砸帝国权威**（§4）+ 给诸侯世俗化教产 |
| **三十年战争** | 帝国权威崩 + 宗派对立越线 → **30YW 局势**（doc 40 §6.3）：皇帝+天主教联盟 vs 新教同盟+外援（瑞典/法国） |
| **威斯特法利亚** | 局势结算 → **诸侯得完整主权、帝国权威永久架空**（doc 40 §6.4）→ 成员变近代主权国家 |
| **反事实岔路** | 若皇帝打赢、权威保持高位 → **中央集权的哈布斯堡普世帝国**（史实没走成的那条路） |

> 神罗让「封建拼图 → 主权国家」这条游戏论题**可玩**：你既能当皇帝试图集权，也能当诸侯争主权。

## 9. 1337 开局（丰富的起点）

- 当前数据：神罗君主为**路易四世·维特尔斯巴赫（巴伐利亚）**，1337 史实如此——且他正被**阿维尼翁教皇（受法国控制，doc 40）绝罚**、与教廷激烈冲突。
- 落地：初始皇帝 = 巴伐利亚（维特尔斯巴赫），**被绝罚、皇权-教权冲突**；哈布斯堡（奥地利）为崛起中的强力竞争者。开局即一台戏。

## 11. 教皇/哈里发回填为原语实例

doc 40 的教皇/哈里发改写为 `world.structures.papacy / caliphate`（type:"religious"）：head=教皇、authority=教皇权威、members=天主教国、selection=枢机/控制权、actions=绝罚/十字军。**与神罗共用同一份结构代码**——这就是 C4「可复用」的兑现。

## 14. 风险

| 风险 | 缓解 |
|---|---|
| 滑向 EU4 的 300 小邦 | 成员封顶 ~20；唯一这种领土结构 |
| 拆国工作量（数据+地块+AI） | 单列数据阶段；部分成员已存在；AI 复用启发式 |
| +20 实体拖累 AI/性能/sim | 成员用轻量国家状态；带默认 getter 防崩；invariants 守门 |
| 帝国政治对新手过载 | 非神罗玩家只需把神罗当一个「松散集团」感知；帝国面板渐进展开 |
| 反事实集权岔路破坏历史感 | 它是「代价」的一体两面——史实分裂 vs 未竟集权，皆有合法路径 |
| 与 doc 40 弧线耦合过深、改残碰崩溃轨 | 复用 35/P1-2 守卫；阶段化落地、每阶段 invariants 5×200 |
