# 39-科技/发展系统设计

> 承接 `36`（国家，废改革槽后把「能力深度」轴指派给科技）、`37`（经济）、`38`（外交与军事）后的设计讨论结论。
> 科技 = **横贯所有系统的「能力深度」脊梁**：决定「能不能」，制度决定「选不选」。结构为 **领域科技树 × 研究流 × 扩散**。
> 日期：2026-06-27

# 第一部分 · 设计总述

## A. 现状的病

| 现状 | 病 |
|---|---|
| 9 项扁平 on/off 技术 | 无树、无路径、无取舍——迟早全点满，`ageProgress` 只是计数（死清单病） |
| 早期无可研究技术（最早火炮 1370） | **开局到 1370 前科技线是空的** → 报告 33「40 季 0 采纳」的真因 |
| ideas 流单薄、印刷是思想主源却又晚又贵 | **鸡生蛋**：想加速思想要先有印刷，印刷却要 1450+ 和 40 思想点 |
| doc 36 把能力轴指派给科技 | 9 个孤立解锁**撑不起一条能力轴** |
| 传播扩散机制存在但只用来 gate 采纳 | 浪费了一个天然的**反雪球均衡器** |
| 技术效果散落硬编码在 economy/warfare/trade | 没有「科技→制度/经济/军事/纪元」的统一接线 |
| 探索是独立小轨、技术史质感薄 | 漏掉整片领域（农业/行政/财政/文化科学） |

## B. 核心思想：领域树 × 研究流 × 扩散，具体不抽象

> **科技是能力深度脊梁，但吸取改革槽的教训——不要抽象层级。** 领域只是组织，真正的单位是**具体的、历史的、有真实效果的技术节点**；「深度」从你点了哪些具体节点里**涌现**。

- **领域树**：5 个领域，每个是一棵 DAG（继承/分叉/汇聚/少量跨领域），≈25 个具体技术节点。
- **研究流**：每领域从**自己的社会活动**自累积研究（B 方案），玩家用**焦点杆**偏向 + 分叉处选。
- **扩散**：技术在接触的国家间传播，落后国靠接触**追赶**——反雪球。

## C. 关键设计决策（讨论结论，已定）

### C1. 研究投入 = per-领域自动累积 + 焦点杆（选定 B）

每领域从其社会活动自生研究（军事←战争/军营、经济←市场/城市、文化←大学/印刷…），沿树半自动推进；玩家只在**分叉处选** + 一个**焦点杆**（指定 1 优先领域：+50% 该领域、其余 −15%）。

> 选 B 而非「单池+手选」的理由：**让五个政体原型在科技上真分化**（商业海洋共和国自然在经济/航海领跑、军事君主国在军事领跑）→ 治「国家同质化」；契合**「研究是社会自生的流、玩家扰动」**总哲学；微操低。雪球风险由扩散对冲。

### C2. 领域科技树（继承关系）

每领域 DAG：技术有 `requires` 前置（**继承**）、**分叉**（多支选一=机会成本）、**汇聚**（高级技术需多前置）、**少量跨领域前置**（把五领域织成网）。除前置外仍卡**纪元（年）+ 传播度**。

### C3. 纪元 = 世界时钟，科技 = 能否用上该纪元的能力

纪元由年驱动（历史照常翻篇，doc 不改）；**科技决定你能不能用上该纪元的高级能力**——落后国到了绝对主义纪元但没「常备军操典」技术，就用不了常备军制度。**不让纪元由科技驱动**（否则落后国卡旧纪元，加剧 13× 鸿沟）。

### C4. 扩散 = 反雪球均衡器

技术被越多接触国掌握 → 你的 awareness 涨越快 → **落后国研究成本越低**（追赶）；先驱付全价（开拓成本）。天然收窄强度鸿沟。

### C5. 修复早期空转真空

每领域补一项 **1337 奠基技术**（成文法典/板甲/三圃制/罗盘海图/大学），**开局即有科技决策**，不必等 1370。

## D. 接缝 + 回到核心循环

> 基底（人口/城市/建筑/贸易接触/稳定）→ **各领域研究流** → 沿树解锁技术 → **解锁制度(36)/兵种(38)/建筑物产(37)/纪元能力** → 反作用回基底 → 扩散把先进度外溢给邻国 → 循环。

**科技 = 「能不能」，制度 = 「选不选」**——正是 doc 36 画的「能力 vs 结构」分工的落地。

# 第二部分 · 落地蓝图

## 1. 五领域与研究来源

| 领域 | 研究流来源（社会活动自生） | 主要喂哪个系统 |
|---|---|---|
| 行政/治理 administrative | 集权度 + 官僚建筑 + 行政活动 | 国家（解锁制度） |
| 军事 military | 军营/堡垒 + 处于战争 + 军事阶层权力 + 常备军 | 军事（解锁兵种） |
| 经济/农业 economic | 市场/贸易流 + 城市化(人口) + 工坊 | 经济（建筑/物产/产出） |
| 航海/探索 naval | 港口 + 探索 + 沿海地块 | 经济（殖民/商路）+ 探索 |
| 文化/科学 cultural | 大学 + 印刷 + 合法性/稳定 + 教士阶层 | 思想流/合法性 |

## 2. 研究流（B 方案）

```js
country.research = { administrative:0, military:0, economic:0, naval:0, cultural:0 };
country.researchFocus = "military";  // 焦点杆，玩家设
// 每季：各领域 research += domainIncome(world, polity, domain) × focusFactor(domain)
//   domainIncome 由 §1 来源算；focusFactor：焦点领域 ×1.5、其余 ×0.85
```
- 取代单薄的 `ideas` 资源（`ideas` 可保留为「文化领域」的别名或并入）。
- **取舍即决策**：焦点杆 + 分叉选，半自动，无研究队列微操。

## 3. 科技树（数据结构 + ≈25 节点）

### 3.1 数据形态（新 `data/techs.js` → `window.HIFI_TECHS`）
```js
TECHS = {
  gunpowder: { domain:"military", era:1370, cost:50, requires:[], awarenessGate:25,
               effect:{ unlockUnit:"artillery" } },
  standing_drill: { domain:"military", era:1600, cost:80, requires:["gunpowder"],
               effect:{ unlockInstitution:["military","standing_army"] } },     // 科技→制度接缝
  bayonet_volley: { domain:"military", era:1700, cost:100, requires:["gunpowder","standing_drill"], // 汇聚
               effect:{ unitUpgrade:"infantry_firearm" } },
  double_entry: { domain:"economic", era:1400, cost:30, requires:["codified_law"],  // 跨领域前置(行政)
               effect:{ unlockInstitution:["fiscal","direct"], moneyMult:1.1 } },
  oceanic_trade:{ domain:"naval", era:1600, cost:70, requires:["ocean_ships","bill_of_exchange"], // 跨领域汇聚
               effect:{ unlockRoute:"newWorld" } },
};
```

### 3.2 ≈25 节点（5 领域 × 纪元，每领域含 1337 奠基项治真空）
| 领域 | 奠基(1337) | 大发现 | 绝对主义 | 革命/工业 |
|---|---|---|---|---|
| 行政 | 成文法典→等级会议 | 复式记账(1400)→直接征税 | 官僚体系(1500)→绝对主义 | 宪政文牍(1750)→议会主权 |
| 军事 | 板甲重骑 | 火药火炮(1370)→炮兵 | 棱堡(1500)·常备军操典(1600)→常备军 | 刺刀齐射(1700) |
| 经济 | 三圃制→粮/人口 | 水力工坊(1400)·银行汇票(1450)→商业制度 | 股份公司(1600) | 蒸汽动力(1750)·铁路(1820) |
| 航海 | 罗盘海图→沿海贸易 | 远洋帆装(1420)·星盘(1450) | 跨洋贸易(1600)→新大陆·风帆战列舰(1650) | — |
| 文化 | 大学→研究++ | 印刷术(1450)→宗改/传播 | 人文主义(1500)·科学方法(1650)→全领域研究++ | 启蒙(1750)→共和/议会 |

> 跨领域织网示例：跨洋贸易(航海) ← 银行汇票(经济)；科学方法(文化) ← 印刷术 → 全领域研究效率。

## 4. 采纳/解锁机制

```
每季每领域：
  frontier = 该领域树上「前置已满 + 纪元年已到」的可研节点
  当 research[domain] ≥ node.cost 且 awareness[node] ≥ node.awarenessGate：
     若 frontier 仅一项 → 自动解锁、扣 research
     若 frontier 多分叉 → 玩家裁断「研究方向」(轻量,或按焦点/AI 启发式自动选)
  解锁即触发 node.effect（解锁制度/兵种/建筑/物产/商路/产出乘数）
```
- 年代门槛仍在（C3：到年才可研，世界时钟）。
- AI 自动按原型启发式选分叉与焦点（商业国偏经济/航海、军国偏军事）。

## 5. 扩散 = 反雪球（C4）

扩展现 `spreadTechnology`：
```js
awareness[tech] += 1 + 已掌握该技术的接触国数(邻国/贸易伙伴/同盟) + 印刷加成
effectiveCost = node.cost × (1 − min(0.5, laggardDiscount(awareness)))  // 落后国成本递减
```
- 先进国研究**未被人掌握**的前沿付全价；技术普及后**落后国打折追赶**。
- 接触渠道：相邻 + 贸易伙伴 + 外交往来（接 doc 38 关系；战争/禁运阻断传播）。

## 6. 接缝（科技的价值所在）

| 接缝 | 机制 |
|---|---|
| **科技 → 制度(36)** | 技术 `unlockInstitution`：常备军操典→`standing_army`、复式记账/官僚→`direct`/`absolutism`、启蒙→`parliamentary`/共和。**科技=能不能、制度=选不选** |
| **科技 → 经济(37)** | 三圃制→农业建筑/粮、水力/蒸汽→工坊层级、银行汇票→商业制度+资本、新物产需对应技术（火炮需铸造+硝石） |
| **科技 → 军事(38)** | `unlockUnit`/`unitUpgrade`：火炮→炮兵、风帆战列舰→海军、棱堡→防御、刺刀→步兵 |
| **科技 → 纪元** | 纪元=世界时钟；技术决定能否用上该纪元高级能力（C3） |

## 7. 探索折进航海领域

- 罗盘海图/远洋帆装/星盘 → 探索点累积 → 发现新大陆 → 殖民物产/跨洋商路（接 doc 37 命名大商路：新大陆白银/跨洋贸易）。
- 取消独立 exploration 小轨，统一进航海树。

## 8. 早期空转真空修复（C5）

- 5 项 1337 奠基技术 → **开局即有 5 个领域可研**。
- 研究流改由基底自生（§2）→ 不再是「+1/季涓流 + 鸡生蛋」。
- 效果：开局头几年就有「往哪个领域投焦点」的真决策。

## 12. 风险

| 风险 | 缓解 |
