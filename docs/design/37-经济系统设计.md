# 第一部分 · 设计总述（经济模型）

## A. 现状的病

| 现状 | 病在哪 |
|---|---|
| 地块 `good` 字段只是二元乘数 | 30 种历史物产被粗暴塌成「食物类/军事类/其余」三桶，**铁和马在引擎里是同一个东西**；地理特色无从体现 |
| `settleCountry` 直接汲取 与 `processTrade` 命名商路是**两套互不相干的钱** | 贸易路线与地块到底产什么**完全脱节**，贸易不是从生产里长出来的 |
| 建筑是泛乘数（市场×1.4） | 没有「针对哪种物产」的真选择 → 报告 33 的「建设不活跃」 |
| 人口被动恢复 | 人口是产出基底，却只会向 basePopulation 缓慢回血，不随粮食盈亏增减 |

一句话：**现在的经济是「三个抽象资源的加减」，不是「会因地理、市场、政策而流动的机器」。**

## B. 核心思想：三层一台机器

> **物产（基底）→ 按财政模块切成 消费/税收/贸易（流）→ 喂人口/国库/市场/满意（压力层）→ 玩家用建筑/政策干预阀 → 反作用回地块。**

- **自然产出**：每地块一种历史特色物产，产量受地形/人口/建筑/气候影响，下游各异（铁→军需、马→骑兵、谷物→人口…）。
- **市场贸易**：产出价值分三用途——本地消费、交税、贸易；**比例由财政模块决定**（与 `36` 共用同一根杆）。
- **经济建设**：建筑/税收/关税/贸易政策是干预阀。

## C. 关键设计决策（讨论结论，已定）

### C0. 物产形态：选「扁平 + 轻链」，不做「真供应链」（**取舍记录**）

讨论过两套：

| | 扁平+轻链（选定） | 真供应链（否决） |
|---|---|---|
| 机制 | 每地块**直接**产出特色物产，无需投入；成品（呢绒）用**条件乘数**模拟增值（控制羊毛 → 呢绒贸易价值+X%），**不真消耗原料** | 物产分原料/成品，成品**消耗投入**才能产；逐物产产/耗台账、短缺级联、进口缺口 |
| 结算 | `O(地块)` 纯函数，沿用 `settleCountry` | `O(地块×物产)` + 约束求解 + 自动进口 |
| 决策深度 | 中 | 高 |
| AI/测试/性能 | 轻、易测、sim 快 | **重**（AI 曾因读漏字段崩溃，链放大此面）、难测、sim 批跑成本陡升 |
| 失衡风险（当前 13×） | 温和 | **放大**（链完整国滚雪球、缺一环国崩盘） |

**否决真供应链的理由**：当前目标是**收窄** 13× 强度鸿沟、做 sim 基线；供应链的级联短缺逆着走，且把负担砸在最痛的 AI 健壮性与 sim 性能上。本时代最浓的历史质感（羊毛→呢绒、明矾垄断、跨撒哈拉金路）用**轻链条件乘数 + 命名大商路覆盖层**即可表达，不必上全台账。真供应链留作 sim 稳定后的**可选后期深化**。

### C1. 30 种历史物产，少数库存 + 少数特殊

- **3 库存**：粮食 / 金钱 / 军需（+ 资本沿用）。多数物产喂其一。
- **6 特殊物产**：黄金（**直接进账**）、白银（金钱+强通胀）、马（骑兵上限）、谷物（人口增长）、木材（造船/建造）、硝石（火炮）。
- 奢侈品（酒/丝/香料/糖/毛皮/琥珀）统一附带少量阶层满意——**品类特性，非逐物产特殊**。

### C2. 人口：粮食盈余缓增，赤字饥荒**真减但缓**

食物盈余 → basePopulation 缓抬（地形封顶）、人口向上限增长；食物**持续赤字 → 饥荒，人口缓慢真减员**。取代被动 `recoverLand`。

### C3. 贸易：货币中介市场 + **可贸易禁运**

不做以物易物台账。各国把**余量物产按固定价×物价指数卖到抽象市场换钱**，用钱**补缺口物产**；命名大商路作加成覆盖层。新增**禁运**：一国可对另一国禁运，切断两国间的买卖（缺方更疼，是经济外交杆，挂 `外交` 系统）。

### C4. 价格：固定基础价值 + 全局物价指数 + 临时事件修正

每种物产一个固定基础价值（香料 ≫ 谷物）；全局物价指数承载白银通胀（价格革命）；临时事件改修正系数。**不做逐物产动态供需价市场**（保 sim 快、平衡可控）。

### C5. 气候活乘数

物产按历史种死，但产量随地形+**气候实时缩放**（寒冷压谷物/葡萄）→ 让**小冰期**情势能真正咬到粮食。

## D. 与核心循环 / doc 36 的接缝

> **财政模块（doc 36）= 经济市场层的「消费/税收/贸易」切分阀。** 领主代征→留地方多、直接征税→进国库多、商业关税→出口贸易多、游牧无税→定居产出基底大砍。经济系统与政府财政模块**共用这一根杆，必须协同**。

### 1.3 产量公式（沿用并扩展现 `tileOutput`）

```
amount(tile, good) = good.yield
  × max(1, population) × control × devastation × occupation
  × terrainAffinity(good, tile.terrain)     // 物产-地形契合
  × climateFactor(good, tile.climate)        // 活乘数：寒冷压谷物/葡萄（小冰期可咬）
  × buildingBonus(tile, good.cat)            // 针对性建筑
```

- `good` 仍是地块**种死**的历史数据（来自地图种子，需扩词汇到 30 种并按主产区重铺，见 §7）。
- **轻链**：成品如 `cloth`，若本国控制/可贸易到 `chainFrom`(羊毛) → 贸易价值 ×(1+chainBonus)，否则降档。**不消耗羊毛**，结算时查条件。

## 2. 人口模型（C2）

```
foodBalance = 粮产出 − 口粮需求(population × SUBSIST) − 军团口粮维护
if foodBalance > 0:  basePopulation += min(GROW_CAP, foodBalance×GROW_RATE)   // 缓抬上限（地形承载封顶）
                     population → 向 basePopulation 缓增
if foodBalance < 0 持续: population −= FAMINE_RATE（真减但缓）; 触发满意/合法性压力
```

- 取代 `recoverLand` 的被动回血；战争破坏仍压人口（叠加）。
- `developTile`（投资本抬人口上限）保留，与粮食驱动增长并行。

## 3. 市场与贸易（C3 + C4）

### 3.1 三分（= 财政模块切分阀）

每地块入市物产的价值 `v = amount × good.baseValue × priceIndex`，按**财政模块**切：

| fiscal 模块（doc 36） | 本地消费（→满意） | 税收（→国库） | 贸易（→市场卖钱） |
|---|---|---|---|
| 领主代征 | 高 | 低 | 中 |
| 直接征税 | 低 | **高** | 中 |
| 商业关税 | 低 | 中 | **高** |
| 游牧无税 | —（定居产出基底已大砍，靠劫掠/贡赋） |

### 3.2 货币市场清算 + 禁运

- **卖余量**：各国把盈余物产按 `v` 卖到抽象市场 → 金钱（税+贸易部分）。
- **补缺口**：缺的物产（如缺铁）用钱按 `v × 进口加价` 买回，填补军需/库存。
- **禁运**：`embargo(A,B)` → A 的买/卖池中**剔除 B**（A 买不到 B 的特产、B 失去 A 的市场）；缺方更疼。挂 `外交` 系统的新动作，与停战/同盟并列。
- **命名大商路（覆盖层）**：跨撒哈拉金路 / 黎凡特香料路 / 威尼斯丝绸路 / 汉萨波罗的海毛皮琥珀路 /（后期）新大陆白银路——给沿线**转口枢纽**额外收入；战争/占领/禁运抬高 `routeCost`（沿用 trade.js）。

### 3.3 价格

- 物产固定 `baseValue`；全局 `priceIndex` 由白银（强）+黄金（弱）累积推高（价格革命链沿用）；临时事件（饥荒/瘟疫/航路开通）改局部修正。

## 4. 经济建设（干预阀）

- **针对性产业建筑**（治「建设不活跃」）：按品类增产对应物产——农庄(食)/矿场(铁铜锡/金银)/马场(马)/林场(木)/盐场(盐)/葡萄园(酒)/作坊(呢绒等手工)。**1–3 级升级**。
- **通用基建**：市场（抬税收/贸易分成）/港口（海贸+命名航路）/道路（连市场、降贸易成本）。
- **军事**：堡垒（军需+防御）。
- **资本**：贸易盈余喂资本池；资本供 `developTile`（抬人口上限）与高级建筑。

## 5. 军需 / 马 的硬度（C 档 5）

- **军需「软」**：短缺 → 军团组织/士气/补给惩罚（沿用现 shortage 逻辑），不硬锁，防 AI 死锁。
- **马「半硬」**：动员骑兵需**马的产出或可贸易来源**（被禁运/无产地 → 动员不出骑兵）。仅此一个物产做硬闸。

## 6. 黄金 / 白银 / 通胀（C 档 7）

- **黄金** → 直接 +金钱（不走三分）；轻微 `priceIndex`。
- **白银** → +金钱 + 强 `priceIndex`（价格革命主源）。

## 9. 风险

| 风险 | 缓解 |
|---|---|
| 30 物产 → 地图重铺工作量大、需对历史 | 单列数据阶段，与层1 并行；先用现有词汇映射、再逐区精修 |
| 市场重写放大 13× 鸿沟 | 卡 sim 之后；用「旧默认产出≈基线」回归断言守门 |
| 禁运被 AI 滥用致全图断贸 | AI 禁运预算 + 仅对敌对/战争国施加；带默认 getter 防崩 |
| 饥荒减员与战争减员叠加过狠 | FAMINE_RATE 取缓；与破坏减员共用节流闸 |
| 轻链条件乘数被误解成真供应链 | 文档 C0 明确「不消耗原料」；UI 标注「需羊毛产地」为加成而非前置 |
