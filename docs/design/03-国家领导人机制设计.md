# 03 - 国家领导人机制设计

## 目标

让国家不再只是资源与政体容器，而是由具体领导人推动。领导人的能力决定国家每季度获得的行政、外交和军事行动点；死亡、政变、继承与选举会改变领导人，并为后续外交互动提供人物入口。

## 时间

- 1337 年第一季度开局。
- 1 回合等于 1 个季度，4 回合等于 1 年。
- 游戏不设固定终局。
- 领导人的年龄、任期和历史换代节点按季度推进。

## 领导人数据

每个国家状态保存独立的 `leader`：

- 姓名、家族、头衔。
- 出生年份、上任年份、历史离任年份。
- 行政、外交、军事能力，范围 0–6。
- 继承制度：世袭、终身选举、定期选举或教会选举。
- 选举任期和当前任期结束时间。
- 历史继任人队列。

1337 年的初始领导人使用真实历史人物。历史继任人耗尽后，系统根据国家文化、家族和政体生成新领导人，不额外标记。

## 政体与头衔

| 政体 | 常见头衔 | 换代方式 |
|---|---|---|
| 君主制 | 国王、公爵、伯爵、大公、苏丹、贝伊、大公爵 | 死亡、退位或政变后继承 |
| 帝国制 | 皇帝、汗 | 死亡、废黜或政变后继承 |
| 商业共和国 | 总督 | 定期或终身选举 |
| 共和国 | 执政官、王公、市长 | 定期选举 |
| 神权国 | 教皇、大团长 | 终身选举 |
| 部族联盟 | 大汗 | 家族继承与部族推举混合 |

具体国家可以覆盖政体默认头衔和任期。

## 行动点

统一行动点拆分为三类：

| 能力 | 每季度行动点 | 用途 |
|---|---|---|
| 行政 | `1 + floor(行政能力 / 2)` | 建设、修路、任命行政官、整合、行政改革 |
| 外交 | `1 + floor(外交能力 / 2)` | 议会协商，以及后续贸易和外交 |
| 军事 | `1 + floor(军事能力 / 2)` | 征兵、移动、军事改革与战争 |

三类行动点各自累计，上限 10。现有操作按用途消耗对应行动点。

## 换代

采用历史节点与动态模拟混合：

1. 未发生重大偏离时，优先在历史离任节点启用真实历史继任人。
2. 高龄、低合法性和强势阶层会增加提前死亡或政变风险。
3. 正常世袭优先使用历史继承人；历史名单耗尽后，生成同家族继承人。
4. 定期选举在任期结束时产生三名候选人。
5. 玩家国家由玩家选择候选人；其他国家按能力、阶层支持和国家局势自动选择。
6. 终身选举政体在领导人死亡后触发选举。

## 界面

国家详情窗口增加领导人区域：

- 头衔、姓名、家族、年龄和在位时间。
- 行政、外交、军事能力及其季度行动点产出。
- 继承或选举制度、任期信息。
- 玩家和外国国家均可查看。

玩家国家发生选举时弹出三名候选人；候选人卡展示能力、家族和支持阶层，必须选择后才能继续。
