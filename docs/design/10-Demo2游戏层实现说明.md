# 10 - Demo2 游戏层实现说明（P0+P1）

结论：**《09-游戏机制完整策划方案》的 P0 全部 8 项与 P1 全部 6 项已在新文件 `prototype/demos/帝国的代价-微信小游戏demo2.html` 中实现并通过验证。原 demo 未改动，可对照游玩。**

## 文件与运行

| 项目 | 说明 |
|---|---|
| 新 demo | `prototype/demos/帝国的代价-微信小游戏demo2.html`（单文件，约 8400 行，原 7545 行基础上叠加游戏层） |
| 原 demo | `prototype/demos/帝国的代价-微信小游戏demo.html`，未改动一行 |

## P0 实现对照

| # | 方案条目 | 实现 |
|---|---|---|
| 1 | 数值重做 | 合法性 0-100（开局 62）；阶层满意度 ±100（事件 ±4~25）；库存/成本 ×10 量级；**贵族权力不再参与任何死亡判定** |
| 2 | 废瞬死 | 四级不满阶梯（安定→不满 -30→抗争 -55→公开对抗 -75）；进入抗争弹**红色事件卡**（让步 40 金 / 弹压 30 军需 / 搁置）；对抗持续 3 季触发**真实叛军军团**（在该阶层主导地块起兵、向都城进军、走现有战争结算）；灭亡仅两途：领土尽失，或首都沦陷+合法性<20 持续 4 季 |
| 3 | 危机重做 | `CRISES` 按年取模表已删除，改为**情势引擎**：黑死病（1345 预兆、1347 登陆按纬度波次北传、检疫减半、教会得势）、小冰期（周期性、寒带减产）、商贸繁荣（市场国受益）；全部作用于**所有国家** |
| 4 | 朝会 | 每季结算后自动弹出：三级预警（红/黄，点击跳转对应面板）、上季国情（每项资源变化的**因果分解**）、三条顾问谏言（太傅/司农/御史/礼部/枢密/中书按国情生成）、使命议程进度、天下事 |
| 5 | 结算预览 | 「结束季度」先弹**御览**：下季四资源预算分项、合法性走向、阶层情绪、战争清单，确认后才结算；点击 HUD 资源条同样打开 |
| 6 | 教学链 | 10 条「太傅课业」（主持朝会→巡视→建设→敕令→议程→使节→御览→改革→整合→粮食盈余），完成给奖励，嵌入朝会与议程面板 |
| 7 | 国家史诗 | 终局（或随时点「收官立传」）：四轴评分（疆域/富庶/制度/威望）、编年史时间线、死因解剖（灭亡时）、史官曰+下一局国家推荐 |
| 8 | 因果日志 | `adjustEstate`/`changeLegitimacy` 全部调用点带 reason；季度报告记录每笔变化来源；旧 `showEnd` 弹窗改为史诗画面 |

| 时代使命 | 首批 10 国 × 2：法兰西（驱逐英人/王权稳固）、英格兰（兵临巴黎/羊毛财富）、威尼斯、奥斯曼、拜占庭、卡斯蒂利亚、神罗、教皇国、金帐、波兰；条件可判定、奖励含永久修正 |

| AI 内政 | 每国每季按优先级：安抚最不满阶层→粮荒复员→建设（沿用建筑优先级）→整合低控制区→低合法性巡游；战时自动动员（上限 3 军团） |

| 经济曲线 | 产出保持地块刻度、库存×10；**军队维护 = 兵力/40**（4000 人吃掉约半个法兰西的季度粮产，养兵第一次真正有代价）；粮仓上限 300+农庄×30；年度人口增长（粮足国家高控制地块+1 农民） |

## v2 界面与文案调整（2026-06-11 第二轮反馈）

| 反馈 | 处理 |
|---|---|
| 术语过于中国风 | 全面改用欧洲史标准译名：朝会→**御前会议**（散会）；顾问改为中世纪宫廷官职——宫廷导师（教学）、王室总管（粮政）、内务大臣（阶层）、御前司铎（宗教合法性）、王国元帅（军务）、大法官（议程）、司法长官（地方治理）、宫廷制图师；敕令改名：武备税→**盾牌钱**（scutage）、御前比武→**比武大会**（tournament）、召对群臣→**王室觐见日**、开仓赈济→**王室施赈**；议程卡：广积粮→充实粮仓、整顿吏治→王室巡察、枕戈待旦→整军备战、敬天法祖→虔诚立国；收官立传→**编年史册**、史官曰→**编年史家曰**、国祚断绝→王统断绝、天下事→列国纪闻。保留本就源自欧洲史的译名（敕令=edict、巡回法庭=eyre、加冕巡游=royal progress、封港检疫=quarantine） |
| 左上选项太多（12 个） | 压缩为 **7 个**：国家 / 敕令 / 议程 / 外交 / 军事 / **政制**（合并改革+议会+决议+阶层）/ **纪闻**（合并事件+日志）；御前会议入口移至顶栏金色按钮 + 国家面板内"御前会议"动作（移动端顶栏隐藏时仍可达，且每季自动弹出）；旧面板名通过 `PANE_ALIASES` 映射，所有跳转链接不受影响 |
| 右上数值两排 | 压缩为**单排**：行动点三格（去掉"季度行动点"文字标签）+ 粮食/金钱/军需/合法性/民心五格同行，字号与间距同步收紧 |
