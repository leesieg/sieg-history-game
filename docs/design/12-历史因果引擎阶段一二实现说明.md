# 12 - 历史因果引擎实现说明（阶段一+二）

结论：**11 号方案的阶段一（引擎骨架）与阶段二（君堡垂直切片）已在 `prototype/demos/帝国的代价-微信小游戏demo2.html` 上全部落地。范式案例"君堡陷落→商路改道→地理大发现→白银通胀"七跳因果链已可端到端运行，且在无人值守模拟中会自发涌现：奥斯曼 1390 年发动海峡之战，围攻狄奥多西城墙九年后于 1399 年攻陷君堡，大西洋四国随即开始探索。13/13 测试通过。**

## 阶段一：引擎骨架

| 系统 | 实现 |
|---|---|
| 贸易路线 | 9 条路线（黎凡特/黑海/红海/汉萨/香槟-莱茵/大西洋沿岸/马格里布金路 + 锁定的好望角/新大陆线），节点锚定到地图城市，图外端点（东方/撒哈拉/美洲）按待定决策 #1 不做实体地图。每季结算：`成本 = 1 + Σ(关税/战争/占领/瘟疫/残破)` → 同池路线按 `权重/成本` 再分配流量，池总量随平均成本萎缩 → 节点收入入国库并写入季报（"商路·黎凡特线（威尼斯）+22"） |
| 关税档位 | 商路面板三档（低/常规/高）：低关税养流量惠商人，高关税单位抽成 ×1.5 但推高路线成本逼流量改道——"奥斯曼选择"的玩家化。调整冷却 4 季、消耗行政点、登记决策台账 |
| 压力仪表 | 四表常驻国家面板与预警：商路依赖（商路收入占比）、军事竞争（接壤最强邻比值+战时）、财政压力（常态支出/收入）、新航路冲动（东方商路成本指数 × 大西洋区位 + 大发现冲动） |
| 决策回响 | 决策台账（敕令/议程/改革/决议/关税/采纳进步均登记）+ 持久回声日志（阶层/合法性变化独立于季报留存）→ 御前会议「决策回响」栏：「上季你定下《盾牌钱》→ 商人满意 -8」 |
| 垂帘听政（速演） | 常例自动结算直到中断：红色事件/情势变化/我国战争/选立领导人/都城告急/期满 40 季（待定决策 #3 默认三类中断已含）。结束弹「摄政报告」：唤醒缘由+期间收支+大事列表。浏览器实测 10 季瞬时完成，被小冰期预兆准确唤醒 |
| 贸易地图模式 | 地图模式栏新增「贸易」：底图弱化、节点地块鎏金；每条路线一条彩线（**线宽=流量、虚线=受阻**），图外端点入图标注（东方/撒哈拉/美洲/好望角航路）；节点圆点大小=该节点季收益、金边=本国节点，悬停显示明细，点击航线直达商路面板；图例实时显示各线流量。开局即初始化贸易快照，第 1 回合可用 |

## 阶段二：君堡垂直切片

| 环节 | 实现 |
|---|---|
| 奥斯曼战略驱动 | `STRATEGIC_DRIVES`：1389 年起（或 1370 年起且扩张到 14 块的霸权路径）对拜占庭发动「海峡之战」，战争目标=君堡，绕过常规宣战的态度/全局战争数限制；战时动员上限提升到 5 个军团 |
| 狄奥多西城墙 | 君堡地块围城进度减半——城墙逼出长期围攻（模拟中围了 9 年）；炮兵参战提供攻城加成（火药改变攻城，正是其历史角色）；海峡之战免于 AI 间 20 季自动和谈（围城战需要时间） |
| 君堡保留地 | 开局"尼科米底亚围城"的战争目标修正回比提尼亚方向（修复了君堡 1339 年就被和谈割走的失真）；AI 常规边境战争不得以君堡为目标——它只属于战略驱动或玩家 |
| 转折情势 | 条件触发（君堡控制者 ≠ 拜占庭，不看年份）：全屏转折卡「第二罗马的陨落」+ 四跳因果链预告；黎凡特/黑海 12 季"易主动荡"成本冲击；AI 征服者默认抬高关税（史实行为）；`discoveryImpulse` 点亮全图大西洋国家的探索压力 |
| 进步扩散 | 复活科技死端：火药武器（战争接触/年份窗口知晓，采纳花钱+贵族满意 -5，解锁炮兵动员）、远洋帆船（2 港口或海洋改革，商人+6）。采纳是政制面板里的决议卡，不是科技树；AI 富裕时自行采纳（模拟中 31 国扩散） |
| 探索里程碑 | 马德拉(10)→几内亚(25)→黄金海岸(40)→好望角(60，解锁好望角线)→印度航线(85，流量升级)→西方大陆(110，解锁新大陆线+白银)。点数来源：《资助航海》敕令+5、压力≥40 时被动+1、「向未知之海」议程+2/季；**远征节奏**：里程碑间隔 ≥6 季（一次远航一年半） |
| 航线开辟 | 首达者成为 patron，其最佳大西洋港口成为终端节点（份额 0.3-0.38）；好望角线加入东方池直接抽走黎凡特/黑海流量——威尼斯结构性衰退由流量物理再分配产生，不是脚本 |
| 白银通胀 | 待定决策 #2 的简化模型：新大陆线收入标记白银 → `物价指数 +白银/600`（上限 2.0，无白银缓慢回落）→ 地块金钱产出 ×(1/指数)，季报出现「物价侵蚀 -X」——拥金而衰每季可见 |
| 因果链卡 | `pushWorldEvent` 支持 chain 数组：纪闻条目可展开「为什么？」逐跳溯源；转折卡正文即因果链 |
| 配套内容 | 奥斯曼新使命「罗马的继承者」；威尼斯「保卫地中海贸易」议程（东方成本 ≥1.2 时出现）；大西洋国家金边议程「向未知之海」 |

## 验收结果（对照 11 号方案）

| 验收项 | 结果 |
|---|---|
| 威尼斯玩家能看见并干预商路收入 | ✅ 商路面板：本国收入/依赖度/物价 + 路线卡（流量/成本/成本因素 chips）+ 关税三档。浏览器实测黎凡特成本因素实时显示「君士坦丁堡卷入战争 +15%」 |
| 任意决策下回合有回响 | ✅ 实测：「上季你定下《盾牌钱》→ 商人满意 -8」 |
| 速演稳定 | ✅ 五轮垂帘合计 25+ 季无崩溃，情势预兆准确唤醒 |
| 七跳链端到端 | ✅ 脚本化测试：陷落→黎凡特流量 -30%+→威尼斯收入 37→29→21（好望角后）→葡萄牙探索压力 +15 以上→远洋帆船→好望角/新大陆解锁→葡萄牙物价 1.56 且季报可见侵蚀 |
| 可偏转（不陷落对照） | ✅ 未陷落时葡萄牙探索压力 16（低位）；仅商路成本上升（不依赖陷落事件）也会制造压力——压力来自成本而非剧本 |
| 历史自发涌现 | ✅ 290 回合无人值守：1390 宣海峡之战→1399 君堡陷落→1400 年代大西洋四国探索竞赛、31 国采纳进步——**没有一行脚本指定这些年份** |
| 回归 | ✅ 13/13 测试（8 旧战争 + 5 新游戏层/因果），原 demo 未动 |

## 数值与时间线（当前调校）

模拟世界的历史比真实压缩约 40-60 年（黑死病 1347 准时、君堡 ~1399、好望角 ~1405），内部因果间距与真实历史同构（陷落→绕航约 6-15 年）。调校旋钮：`STRATEGIC_DRIVES.minYear`（1389）、城墙减半系数、远征间隔（6 季）、`物价/600`。

## 已知边界（留给阶段三+）

- 信仰纪元（印刷→宗教改革扩散）未开始——扩散引擎接口已就绪（advances 的 aware/can/apply 三段式）
- 纪元状态机未实装：君堡转折已是事实层面的"纪元翻页"，但无纪元卷轴 UI 与规则批量切换
- 探索目前必经好望角线后才开新大陆（线性链）；西航分支决策未做
- 压力层尚未接入 AI 宣战权重（奥斯曼用专门驱动；通用"压力→战略"映射属 LLM 阶段输入）
- 商路依赖压力对玩家法兰西常年为 0（法兰西无节点城市）——符合史实但可考虑给香槟集市加节点
