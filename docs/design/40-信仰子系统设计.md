# 40-信仰子系统设计

> 承接 `36`（国家，把宗教从政体剥离）、`37`（经济）、`38`（外交军事，圣战 CB 来源指向此）、`39`（科技，印刷→宗改）后的设计讨论结论。
> 信仰不是一个边角子系统——它承载《帝国的代价》的**主论题**：**宗教改革 → 三十年战争 → 威斯特法利亚 → 近代主权国家**（中世纪基督教世界 → 主权国家体系）。
> 日期：2026-06-27

---

# 第一部分 · 设计总述

## A. 现状的病

| 现状 | 病 |
|---|---|
| 宗教 = 地块标签 + 给教士添堵 | 极薄、半装饰；征服异教地块在宗教上什么都不发生 |
| 有「信仰压力」却**没「信仰流」** | 信仰是六大流里唯一**没有流动机制**的；改宗是「瞬间全翻」非过程 |
| 信仰压力写死「非天主教」 | 多宗教世界算不对 |
| 宗教改革 = 1517 一个 flag + 一条因果链 | 没有「萌芽→扩散→对立→战争→和约」的历史过程 |
| 无宗教权威/跨国维度 | doc 38 的圣战 CB **没有来源** |
| doc 36 把宗教拽出政体却没给它家 | 四散：地块字段+法律+压力+flag |
| 宗教地理种子薄（多为「异教」，对不上 1337 史实） | 需像物产一样重铺 |

## B. 核心思想

> 用与科技同范式：**信仰是一条传播驱动的流**，玩家在关口扰动它；它反作用回合法性/阶层/战争；并由一条**历史弧**把游戏推向近代国家。

四组件 + 一弧线：**两级宗教 · 信仰流(传播/皈依) · 重型宗教权威 · 宗教政策杆 · 历史弧(宗改→30YW→威斯特法利亚→近代国家)**。

## C. 关键设计决策（讨论结论，已定）

### C1. 两级宗教：信仰组 / 信仰
- **信仰组**（基督教/伊斯兰/犹太）：**跨组 = 异教徒 → 十字军/圣战**（总体战、目标征服皈依）。
- **信仰**（天主/东正/路德/归正；逊尼/什叶）：**同组跨信仰 = 异端/分裂 → 宗教战争（基督教世界内部）**。威斯特法利亚终结的正是**组内**宗教战争。
- 跨组少数比跨信仰少数更难整合、不满更高；权威按信仰组织（教皇只领天主教）。

### C2. 重型宗教权威（actor 级跨国机构）
天主教·教皇做成元层 actor：**教廷控制权竞争**（1337 阿维尼翁，受法国控制）、**绝罚**、**十字军号召**、**信仰捍卫者**、**教皇虔诚度**。伊斯兰·哈里发、东正教·牧首同构（东正较轻）。

### C3. 逐地块信仰流 + 传教皈依
宗教在地块上缓慢传播（像科技扩散）；玩家可**传教**转化少数地块抬统一。

### C4. 宗教政策杆（从政体移出）
宽容/正统/强制传教 → 喂教士阶层满意（doc 36 残留耦合）+ 少数不满 + 统一速度。

### C5. 历史弧：窗口脚本化、结局涌现
宗改窗口按纪元必开；**是否爆发 30YW、和约怎么签由实况决定**，非固定日期硬演。30YW 用**局势引擎**装，威斯特法利亚是其**历史结算结局**。

### C6. 世俗化红利 + 诸侯改宗动力
改宗夺教产 = 一次性国库 + 永久土地/收入；神罗诸侯改宗还**摆脱皇帝+教皇权威**得自治——改宗是理性的物质+主权选择。

## D. 接缝 + 回到核心循环

> 信仰流（传播/统一/张力，压力层）→ 改宗/传教/十字军（关口扰动）→ 合法性/教士/教产/CB（反作用）→ 历史弧把世界从基督教世界推向主权国家 → 循环。

# 第二部分 · 落地蓝图

## 1. 宗教数据（两级 + 重铺）

```js
// data/faiths.js → window.HIFI_FAITHS
FAITH_GROUPS = { christian:{label:"基督教"}, islam:{label:"伊斯兰教"}, jewish:{label:"犹太教", minority:true} };
CONFESSIONS = {
  catholic:{ group:"christian", label:"天主教", authority:"papacy" },
  orthodox:{ group:"christian", label:"东正教", authority:"patriarchate" },
  lutheran:{ group:"christian", label:"路德宗", authority:null, emergesAtEra:"faith" },
  reformed:{ group:"christian", label:"归正宗", authority:null, emergesAtEra:"faith" },
  sunni:{ group:"islam", label:"逊尼派", authority:"caliphate" },
  shia:{ group:"islam", label:"什叶派", authority:"caliphate" },
};
tile.confession = "catholic";              // 组由 CONFESSIONS[x].group 派生
tile.faithStrength = 80;                    // 当前信仰牢固度 0-100
country.stateConfession = "catholic";
country.faith = { piety:60, papalFavor:50, policy:"orthodoxy", secularized:false };
```
> **宗教地理需按 1337 史实重铺**（天主：西/中欧；东正：拜占庭/塞尔维亚；逊尼：奥斯曼/马穆鲁克/格拉纳达/北非）——单列数据阶段，同物产。

## 2. 信仰流与皈依（补缺失的流）

```js
// 每季 spreadFaith：地块信仰按邻接+国教施压缓慢转化（类 spreadTechnology）
tile.faithStrength 受：周边同信仰地块数、本国国教施压(policy)、宗教权威半径、印刷(新教)
若异信仰且 faithStrength 跌破阈值 → 地块 confession 翻转
// 玩家动作：传教 missionary（投入金钱/外交点）→ 对目标少数地块加转化压力
// 统一 unity = 国教地块占比；跨组少数惩罚 > 跨信仰少数
```
- **信仰压力一般化**：`pressures.faith = 非国教地块加权（跨组×重、跨信仰×中）+ 宗改张力`，取代写死的「非天主教」。

## 3. 宗教政策（从政体移出的杆）

| policy | 教士阶层 | 少数地块 | 统一 |
|---|---|---|---|
| 宽容 tolerance | 满意↓ | 不满↓、稳 | 慢 |
| 正统 orthodoxy | 满意↑ | 不满↑ | 推进 |
| 强制传教 conversion | 满意↑ | 大不满+叛乱风险 | 快转化 |

> 喂 doc 36 保留在国家侧的教士/教会阶层（church/clergy/imperial_church）满意度——这是剥离后的残留耦合。

## 4. 宗教权威（重型）

```js
world.faith = {
  papacy:{ head:"教皇国", controller:"法兰西王国" /*阿维尼翁*/, defender:null, crusadeTarget:null, council:false },
  caliphate:{ head:"马穆鲁克苏丹国", ... },
  patriarchate:{ protector:"拜占庭帝国", ... },
};
```
- **教廷控制权竞争**：天主教列强靠捐献/势力争 `controller`；控制者获认可战争、合法性、什一税分成。
- **绝罚 excommunicate**：教皇/控制者绝罚某天主教君主 → 合法性重击 + 其他天主教徒获讨伐 CB + 圣事中止。
- **十字军号召 callCrusade**：设 `crusadeTarget`（异教/异端）→ 信徒获便宜 CB + 回报（虔诚/合法性/赎罪券钱）。
- **信仰捍卫者 defender**：授予领头天主教国 → 威望+领导。
- **教皇虔诚度 papalFavor**（每天主教国）：十字军/捐献↑；攻击教皇/同教内战/世俗化↓；过低 → 绝罚风险。

## 5. 圣战 / 宗教 CB（→ doc 38）

| 烈度 | 条件 | 性质 |
|---|---|---|
| 十字军/圣战 | **跨信仰组**（基督 vs 伊斯兰）或教皇号召 | 总体战、征服/皈依、低声誉代价、高虔诚回报 |
| 宗教战争 | **同组跨信仰**（天主 vs 新教/东正） | 组内、可和谈、威斯特法利亚后**失效** |

> 现成舞台：伊比利亚收复失地、巴尔干抗奥斯曼、波罗的海十字军（条顿骑士团）、1054 东西大分裂、地中海前线。

## 6. 历史弧（centerpiece）

复用 纪元/因果链(history) + 局势引擎(struggle) + 集团(doc 38) + 政体(doc 36)，**窗口脚本化、结局涌现**。

### 6.1 宗教改革（信仰纪元 1517 开窗）
- 触发：信仰纪元 + 印刷(doc 39) + 累积**宗改张力**（教廷腐败/赎罪券/低改革）。
- 机制：路德宗/归正宗作为新信仰从神罗萌芽 → **沿信仰流扩散**（印刷加速）。
- 扩展现有 `reformation_split` 因果链为完整起点。

### 6.2 诸侯改宗动力（C6，复用制度抉择裁断机制）
每个天主教国/神罗诸侯反复弹「**改宗抉择**」：
| 收益 | 代价 |
|---|---|
| **世俗化红利**：夺教会地产 → 一次性国库 + 永久土地/收入（§7） | 绝罚 |
| **摆脱皇帝+教皇权威**（神罗诸侯得自治/主权） | 天主教十字军 CB 目标 |
| 国教改为新教、教随其国 | 皇帝敌意 + 国内天主教阶层不满 |
→ 产生信仰拼图，催生集团（天主教联盟 vs 新教同盟，doc 38）。

### 6.3 三十年战争（做成一个「局势」）
- 用百年战争那套 **struggle 引擎**装：神罗为中心的多方信仰-政治大战；principals = 天主教联盟/新教同盟，interlopers = 哈布斯堡/法国/瑞典/西班牙选边。
- 触发：神罗内高宗派张力 + 集团对立 + 导火索（爆发态）。
- 战中：**破坏放大**（人口巨损）、信仰+政治混合目标、**外国干涉**——天主教法国为打哈布斯堡而支持新教（**国家理由压倒信仰**，近代国家的预兆，建模为干涉者按政治而非信仰选边）。

### 6.4 威斯特法利亚（局势的历史结算结局 → 因果链）
30YW 局势结算（决定性结局或 1648 历史结算）→ 触发 `westphalia` 因果链：
- **教随国定**：各君主 `stateConfession` 锁为主权选择 → 改宗不再招教皇 CB。
- **主权确立**：神罗诸侯得完整主权（doc 36 独立模块化国家）；皇帝权威架空。
- **组内宗教 CB 失效**。

### 6.5 近代国家兑现清单（弧线终点 = 游戏论题）
| 切换 | 内容 |
|---|---|
| 组内宗教 CB 关闭 | 基督教世界内宗教战争失法理；跨组十字军削弱但可残留（抗奥斯曼） |
| 合法性来源转移 | 从教皇认可 → **国家主权**（接 doc 36 绝对/宪政合法性，不再神授/教授） |
| 教皇降权 | 教廷沦为地区性意大利强权，失跨国权威与控制权竞争 |
| 政体走向主权国家 | 加速绝对/议会原型（doc 36） |

> 整条弧 = 信仰子系统 → 因果链开宗改窗 → 集团 → **局势(30YW)** → 结局=威斯特法利亚因果链 → 近代国家规则。**全部复用已有机器。**

## 7. 世俗化红利（教产没收）

- 改宗（或天主教绝对主义与教廷协约）→ **没收教会阶层地产/财富** → 一次性国库 + 教会阶层的土地/权力**再分配**（走 doc 36 `reconcileEstates` 同系回流）。
- 这是诸侯改宗的物质动机，也是信仰↔经济的接缝。

## 8. 接缝

| 接缝 | 机制 |
|---|---|
| 信仰 → 国家 | 统一↑→合法性；政策→教士满意；绝罚→合法性重击；威斯特法利亚后合法性转国家主权 |
| 信仰 → 军事(38) | 十字军/宗教 CB；教皇号召十字军给信徒便宜 CB |
| 信仰 → 经济(37) | 教会持地产/什一税；**世俗化红利**（§7） |
| 信仰 → 科技文化(39) | 印刷→宗改扩散；大学/教士→文化研究 |
| 信仰 → 外交(38) | 同信仰亲和；天主教联盟/新教同盟集团归边 |
| 信仰 → 转折 | 信仰纪元=宗改窗；威斯特法利亚因果链→近代国家 |

## 10. 分阶段落地

| 阶段 | 内容 | 排序 |
|---|---|---|
| **两级宗教 + 重铺 + 信仰流** | faiths.js、地块两级、spreadFaith/传教、统一、压力一般化 | 较早可做（触碰教士/合法性，中等风险） |
| **宗教政策杆** | tolerance/orthodoxy/conversion → 教士/少数/统一 | 随上（接 doc 36 教士耦合） |
| **重型权威** | 教皇控制权/虔诚/绝罚/十字军/捍卫者 | 加法跨国层 |
| **圣战 CB** | 跨组/跨信仰 CB（接 doc 38） | 依赖 doc 38 |
| **历史弧** | 宗改诸侯抉择 + 30YW 局势 + 威斯特法利亚 + 近代国家规则 | 依赖 struggle 引擎 + doc 36(近代国家) + doc 38(集团/CB) |
| **数值调平** | 转化速率、世俗化红利、30YW 破坏、虔诚阈值 | **sim 之后** |

> 弧线后段（威斯特法利亚/近代国家）只在长局（1517–1648+）生效，是**终局论题**；早期试玩优先级低，但定义了游戏的最终目的地。

## 11. 风险

| 风险 | 缓解 |
|---|---|
| 弧线变成硬脚本/失去能动性 | 窗口脚本化、结局涌现（C5）：宗改必开、是否 30YW/和约由实况定 |
| 30YW 破坏过狠/雪球 | 复用 struggle 引擎现有节流；外国干涉平衡；sim 调破坏 |
| 重型权威复杂/AI 难管 | AI 虔诚/十字军/控制权启发式；带默认 getter 防崩 |
| 宗教地理重铺工作量 | 单列数据阶段，先粗映射再精修 |
| 强制传教挫败感 | 宽容政策作为缓压出口；少数叛乱有上限 |
| 近代国家关闭宗教 CB 显得突兀 | 它是游戏论题；UI 明示威斯特法利亚的规则切换 |
| 改宗夺教产致教士阶层崩 | reconcileEstates 同系回流 + 权力守恒；天主教国可走协约温和路径 |
