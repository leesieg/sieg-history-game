# 14 - 流层补齐实现说明

结论：**对照 11 号方案逐节审计后，本轮补齐了全部遗漏机制。四个流（贸易/知识/信仰/人口）全部落地，11 号方案除阶段四（LLM 战略层）及三个按设计保留项外已 100% 实装。15/15 测试通过。**

## 本轮补齐清单

| 机制 | 方案出处 | 实现 |
|---|---|---|
| **人口流：劳力分岔** | 5.4（第 4 流） | 黑死病侵袭过的国家解锁互斥决议对：**「顺应劳役松动」**（西欧型结构：城市化/低控制地块 ≥30%——平民 +12、贵族 -8/-4 权力、人口恢复粮 +6/季×16）vs **「再版农奴制」**（东欧型：贵族主导地块 ≥25%——贵族 +10/+6 权力、平民 -12、庄园谷物粮 +10/季×16、全国控制 +4）。AI 在染疫 8 季后按自身结构自动选边。**同一场瘟疫在不同结构里产生相反后果**，两条路都有因果链入纪闻 |
| **资本池** | 5.4 后半 | 商路收入 15% 沉淀为 `capital`（商路面板可见）；**蒸汽机需资本池 ≥150、铁路消耗资本池 250**——"贸易→资本→工业化"的传导链闭合，这正是"为什么先工业化的是商业国家"的机制表达 |
| 波罗的海粮道 | 5.1 路线表第 10 条 | 信仰纪元自动开通（马林堡/罗斯基勒海峡税/布鲁日），解锁事件带因果链："城市化的西欧吃上东欧庄园的谷物"——与再版农奴制决议互为表里 |
| 棱堡工事 | 5.2 进步表 | 发现纪元扩散；被围攻进度 ×0.6——**火药 vs 棱堡的军事革命攻防回合**（实测围攻 12→7/季） |
| 复式记账 | 5.2 | 意大利源点沿商路扩散；金钱产出 +10%、商人权力 +3 |
| 三角贸易体系 | 5.2 | 王权纪元+新大陆线开通后可采纳；商路收入 +25% 并加速资本沉淀 |
| 铁路 | 5.2 | 工业纪元；全国控制 +8、军需 +15%（"铁路动员"） |
| **财政-军事国家分岔** | 5.2/七·王权纪元 | 互斥决议对：**「绝对主义财政」**（核心权力 ≥60：专卖 +15% 金钱、集权 +8、贵族 -8）vs **「议会举债制」**（议会已开+商人权力 ≥38：解锁《发行国债》敕令——+200 金、债息 -12/季×24）。**同一道财政压力题的两个历史答案**；债息计入财政压力仪表，给革命纪元埋下引线 |
| 商人权力挂钩商路 | 5.1 | 商路收入/6（上限 +14）作为**结构项写进商人类阶层权力公式**（实测威尼斯 57→63）——商业共和国的寡头化由贸易兴衰直接驱动 |
| 消费端物价 | 4.1 第③跳 | 东方商路成本 ≥1.35 时，无东方节点的国家贵族每年满意 -1（"东方奢侈品腾贵"）+ 季报事件——**君堡的代价传到巴黎的餐桌** |
| 总督代理 | 9.1 后期减负 | 王权纪元起国家面板可开关：闲置行政点自动整合低控制地块/建设（每季至多 3 次，行为入季报）——后期疆域膨胀不再拖慢回合 |

## 验证

demo2-flows 测试 8 组场景：劳力分岔双路效果、资本池→蒸汽机门槛→铁路扣减、粮道纪元解锁（流量 51）、财政分岔+国债+压力联动、奢侈品惩罚（节点主豁免）、棱堡围攻衰减、总督代理消耗行政点、商人权力结构对照。全套件 15/15。
