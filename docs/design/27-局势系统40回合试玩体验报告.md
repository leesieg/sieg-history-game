# 27 - 局势系统 40 回合试玩体验报告

## 结论

局势系统方向正确，已经让百年战争从“战争列表里的一个条目”变成了有阶段、有阵营、有使命、有历史跨度的主线系统。

但当前版本还没有形成一局好玩的法兰西战役。核心问题是：**局势阶段在转，真实战争却没有跟上；按钮显示可用，执行规则却报错；玩家推进 40 回合后，目标没有实质变化。**

## 关键节点

| 回合节点 | 体验结果 |
|---|---|
| 开局 | 右下角出现“百年战争 · 对峙 · 1337年”，局势入口清楚 |
| 开局局势窗 | 能看到法兰西阵营、英格兰阵营、教皇国居中调停、三段国家使命 |

| 第 8 季 | 百年战争进入“鏖战”，阶段变化有效 |
| 第 12 季 | 黑死病出现，右侧待办出现黑死病 |
| 第 13 季 | 法兰西与英格兰真实战争停战 |
| 第 16 季 | 局势进入“疲惫议和”，但真实战争已不存在 |
| 第 24 季 | 黑死病爆发，人口与产出短暂受影响 |
| 第 32 季 | 百年战争再次进入“鏖战”，但法英仍没有真实战争 |
| 第 40 季 | 局势为“疲惫议和”；“有利停战谈判”显示可用，点击后提示“目前没有与对手的战争可议和” |

## 已经有效的体验

| 体验点 | 说明 |
|---|---|
| 局势入口有效 | 右下角浮窗比藏在军事抽屉里更清楚 |
| 阶段节奏有效 | 对峙、鏖战、疲惫议和的循环能让玩家感到历史进程在推进 |
| 阵营表达有效 | 法兰西、英格兰、苏格兰、勃艮第、弗兰德斯、教皇国等关系清楚 |
| 使命结构有效 | “稳住王国核心 / 收复争议领地 / 争取有利和平”比单一目标更适合战役 |
| 纪闻记录有效 | 世界事件能记录“百年战争进入某阶段” |
| 多局势并存有效 | 百年战争与黑死病能在 40 回合内同时存在，历史模拟感增强 |

## 主要问题

| 问题 | 具体表现 | 影响 |
|---|---|---|
| 局势和真实战争脱节 | 第 13 季法英停战后，百年战争局势仍继续进入鏖战 | 玩家会觉得局势是“假进程” |
| 决议可用态不可信 | 按钮显示可用，但点击后报错 | 破坏玩家对规则的信任 |
| 使命没有推进 | 40 回合后仍停在“收复争议领地：进行中” | 玩家没有成就感 |
| 推荐建议为空 | 局势面板没有稳定给出下一步建议 | 玩家不知道该做什么 |
| 资源压力太弱 | 40 回合后粮食、金钱、军需大量增长 | 战争和黑死病都没有制造取舍 |
| 战争疲惫无感 | 40 回合后战争疲惫仍为 0 | 长期战争没有代价 |
| 黑死病阻塞弱 | “疫病逼近首都”存在，但没有强烈打断感 | 大事件像通知，不像危机 |
| 右侧待办混杂 | 百年战争、黑死病、外交、建设机会混在一起 | 主线优先级不清 |
| 终局太远 | 面板写 1337–1453，但 40 回合看不到阶段性结算 | 小游戏节奏偏慢 |
| UI 行动少于信息 | 局势窗信息很多，但真正有效的行动少 | 看起来像展示面板，不像操作面板 |

## 优化建议

| 建议 | 做法 |
|---|---|
| 修复决议可用态 | 启用按钮前同时检查局势阶段、真实战争状态、资源、外交状态 |
| 同步局势和真实战争 | 局势进入鏖战时，如果法英未交战，应触发再开战、边境冲突或改名为备战阶段 |
| 停战后阶段改名 | 没有真实战争时，不要显示“鏖战”，改成“停战期 / 备战 / 外交拉锯” |
| 每阶段给明确目标 | 对峙阶段拉盟友和备战；鏖战阶段攻占目标；议和阶段谈判或重整 |
| 推荐必须不为空 | 每阶段至少提供 2 条建议，且能执行或能跳转到对应面板 |
| 强化战争消耗 | 鏖战期提高军需、粮食、金钱和 POP 消耗 |
| 接入战争疲惫 | 鏖战持续越久，战争疲惫越高，并影响合法性、产出和阶层满意 |
| 强化黑死病反馈 | 明确展示人口损失、财政救济、产出下降、恢复周期 |
| 做 40 回合阶段结算 | 到 1347 年给一次“十年战局评估”，判断优势、劣势、奖励、惩罚 |
| 待办分组 | 右侧分为主线局势、阻塞裁断、普通机会，避免平铺混杂 |

## 优先级

| 顺序 | 任务 | 原因 |
|---|---|---|
| 1 | 修复局势决议按钮可用态 | 这是规则可信度问题，必须先修 |
| 2 | 让局势阶段和真实战争状态同步 | 这是局势系统是否成立的核心 |
| 3 | 给 40 回合法兰西局做阶段性结算 | 让小游戏节奏内能看到结果 |
| 4 | 强化战争疲惫和资源消耗 | 让长期战争产生真实取舍 |
| 5 | 改造右侧待办优先级 | 让玩家知道先处理主线还是危机 |

## 可验收标准

| 验收项 | 标准 |
|---|---|
| 决议可信 | 所有显示可用的局势决议点击后都能执行，不会再出现状态矛盾报错 |
| 阶段可信 | 没有真实战争时，不显示“鏖战”或不可执行战争型决议 |
| 主线推进 | 40 回合内至少能完成或失败一个阶段使命 |
| 压力存在 | 连续跳 40 回合，资源不能无脑膨胀 |
| 战争代价 | 鏖战期战争疲惫必须上升，并反作用到国家状态 |
| 历史事件有效 | 黑死病必须能让玩家明确感到人口、财政、产出受损 |
| 小结算存在 | 1347 年左右触发一次十年战局评估 |
