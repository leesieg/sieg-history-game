# 06 - 战争与和平系统设计

结论：战争系统采用“外交决定为何开战、地图决定战争代价、和平决定战后秩序”的闭环。玩家操作地图上的军团，但不进入独立战术战场。

> [!important]
> 本设计只调用项目已经存在或在本文中明确定义的机制。不会为局部效果临时增加威望、将领野心、家族影响力、职位或封地等孤立系统。

## 设计目标

| 目标 | 规则 |
|---|---|
| 战争有政治目的 | 每场战争必须拥有主战争目标，和平要求受战争目标与战争分数约束 |
| 地图操作有分量 | 军团逐六边格移动，受地形、控制区、补给、要塞和海峡影响 |
| 胜负不是纯兵力比较 | 战斗计算有效参战兵力、正面宽度、组织度、士气、将领和条件战机 |
| 战争真实伤害国家 | 军事行动先影响地块与 POP，再汇总为产出下降和战争疲惫 |
| 复杂度集中在决策 | 玩家管理军令、编制、路线和补给优先级，不逐项运输粮食或等待征召训练 |

## 总体闭环

```mermaid
flowchart LR
    A["外交矛盾"] --> B["战争借口与战前危机"]
    B --> C["动员与部署"]
    C --> D["逐格移动与地图作战"]
    D --> E["占领、伤亡与地块破坏"]
    E --> F["战争疲惫与战争意志"]
    F --> G["和平谈判"]
    G --> H["领土、条约与从属关系变化"]
```

## 战争生命周期

| 阶段 | 玩家行为 | 系统结果 |
|---|---|---|
| 战前危机 | 提出要求、召集盟友、威慑、让步或宣战 | 确定战争目标、阵营与外交代价 |
| 动员 | 征召阶层兵源、招募佣兵、调整军团 | POP、阶层满意度、金钱和军需开始承压 |
| 部署 | 设置路线、军令、补给基地与优先级 | 形成季度行动计划 |
| 作战 | 统一结算移动、遭遇战、围城和封锁 | 地块控制、伤亡、组织度发生变化 |
| 消耗 | 处理补给、占领、POP 损失和产出下降 | 战争疲惫上升，战争意志变化 |
| 求和 | 提交和平要求或单独媾和 | 战争分数限制可索取内容 |
| 战后 | 复员、恢复生产、执行停战和条约 | 战争疲惫逐步恢复，世界秩序改变 |

## 战前危机与宣战

| 战争类型 | 战前流程 |
|---|---|
| 防御、独立战争 | 立即开战 |
| 有明确宣称的边境战争 | 先进行 1 个季度的最后通牒 |
| 附庸、王位、吞并战争 | 进行 2 至 3 个季度的外交危机，其他国家可以选边 |
| 无理由宣战 | 可以立即开战，但大幅增加威胁、战争疲惫和国际反对 |

每场战争包含：

- 一个主战争目标。
- 若干开战前公开的次要要求。
- 战争期间基于新事实追加的要求。
- 与战争目标无关的和平要求需要支付更高战争分数，并产生更严重外交后果。

## 军队的两个维度

“作战兵种”与“兵源类型”必须分开保存。

### 作战兵种

| 兵种 | 核心作用 | 限制 |
|---|---|---|
| 步兵 | 占据正面、守城、承受主要战斗 | 机动和追击能力较弱 |
| 骑兵 | 侦察、侧翼、追击 | 山地、森林、围城中效率降低 |
| 炮兵 | 远程火力、打击密集阵形、提高围城进度 | 后期科技解锁，维护与补给成本高 |

弓兵、弩兵、长枪兵等不成为独立兵种，而是步兵在不同时代、文化和科技下的单位型号。

工程和后勤不作为独立单位，由科技、改革、道路、港口、军需仓和将领能力提供修正。

### 兵源类型

| 类型 | 兵源 | 特点 |
|---|---|---|
| 核心卫队 | 国家直属 POP | 规模小、战斗稳定、维护昂贵 |
| 职业军团 | 国家兵源 | 长期服役、训练稳定 |
| 常备军 | 国家兵源 | 由较晚科技与改革解锁，补充稳定 |
| 征召兵 | 各阶层 POP | 数量较大，直接影响生产和阶层满意度 |
| 雇佣兵 | 指定城市的佣兵团 | 整团招募，不消耗本国 POP |

一个单位同时拥有作战兵种和兵源类型，例如“贵族征召骑兵”“职业步兵”“雇佣炮兵”。

## 军团编制

军团是地图上的最小操作单位，每支军团由若干步兵、骑兵和炮兵单位组成。

| 项目 | 规则 |
|---|---|
| 单位槽 | 基础 4 至 10 格，由改革和将领指挥能力调整 |
| 拆分与合并 | 只能在未交战、未溃退且补给正常时进行 |
| 指挥上限 | 超过将领指挥能力后降低组织度与协同效率 |
| 移动速度 | 由地形、补给和军团中最慢的主要兵种决定 |
| 地图棋子 | 地图只展示军团，不展示军团内部每个单位 |

## 征兵、阶层与 POP

| 阶层来源 | 主要兵源 |
|---|---|
| 贵族 | 骑兵、核心卫队和部分职业部队 |
| 农民 | 步兵与大规模征召兵 |
| 城市市民 | 城市步兵及后期炮兵兵源 |
| 神职人员 | 特定改革或政体下的步兵、骑兵 |
| 部族、边疆 POP | 骑兵和适应当地地形的步兵型号 |

- 征兵法律规定各阶层可动员比例。
- 玩家按地区、阶层和单位类型分批动员。
- 动员后部队直接在兵源地生成，不设置独立训练或集结倒计时。
- 新部队必须从兵源地逐格前往前线。
- 动员立即减少当地可生产 POP，并影响来源阶层满意度。
- 战斗减员分为死亡、永久伤残、被俘和逃兵；只有死亡与永久伤残直接减少 POP。
- 复员者返回原兵源地。

## 雇佣兵

佣兵以完整佣兵团招募，不逐个购买单位。

| 属性 | 规则 |
|---|---|
| 编制 | 固定步兵、骑兵或炮兵比例 |
| 成本 | 支付招募费与季度军饷 |
| 契约 | 到期后续约或离队 |
| 首领 | 使用佣兵团自带首领，不能随意替换 |
| 忠诚度 | 仅佣兵拥有；欠饷和惨重伤亡会降低忠诚度 |
| 补员 | 只能在指定友好城市付费补充，不消耗本国 POP |

普通军团不使用忠诚度，只使用士气、组织度和补给。征召兵额外受到来源阶层满意度影响。

## 季度军令与逐格移动

玩家每季度为军团规划多格路线，并选择一项主要军令。季度结算时，所有国家的军令同时执行。

| 军令 | 作用 |
|---|---|
| 行军 | 沿路线逐格推进 |
| 防守 | 构筑阵地并强化所在格控制区 |
| 进攻 | 主动寻找敌军接战 |
| 迟滞 | 延缓敌军并尝试保持有序撤退 |
| 围城 | 包围或攻击要塞 |
| 强行军 | 牺牲组织度和补给换取额外移动 |
| 就地征粮 | 获得临时补给，同时破坏当地 POP 和产出 |
| 有序撤退 | 主动离开危险区域并保存较多组织度 |

### 地形移动

| 地图条件 | 移动效果 |
|---|---|
| 平原、道路、己方核心区域 | 移动较快 |
| 丘陵、森林、沙漠、敌方地区 | 移动较慢 |
| 沼泽、积雪、恶劣气候 | 极慢并可能损失组织度 |
| 大河 | 作为地块边界，跨越需要额外移动力 |
| 高山 | 不可通行，只能通过山口 |
| 海峡 | 需要运输能力和局部制海权 |

### 有限控制区

- 成建制军团对相邻地块产生控制区。
- 进入敌方控制区后停止本次移动。
- 离开控制区需要额外移动力并损失少量组织度。
- 不能直接从一个敌方控制区移动到另一个敌方控制区。
- 骑兵比例、侦察优势、撤退军令和将领机动能力可以降低限制。
- 河流、山口和要塞强化控制区；补给中断会削弱控制区。

## 遭遇、战斗与随机性

双方争夺同一地块时触发遭遇战：

- 一方先到，获得布阵和地形选择优势。
- 同时抵达时，由骑兵比例、将领机动和当前组织度决定先手。
- 多支军团抵达时，正面宽度决定实际参战兵力，其余作为预备队。

战斗按六个阶段自动结算：

1. 侦察与布阵。
2. 远程压制。
3. 正面接战。
4. 侧翼与预备队投入。
5. 士气崩溃。
6. 撤退与追击。

胜负比较有效参战兵力，不直接比较军团总人数。

随机性使用有条件的战机事件，不采用无条件的大幅骰子修正。伏击、泥泞、侧翼突破、阵形拥堵、指挥失误等条件可以触发连锁崩溃，使优势方低概率战败。结果必须能由战斗报告解释。

## 士气、组织度与经验

| 状态 | 作用 |
|---|---|
| 士气 | 决定军团是否继续战斗或崩溃 |
| 组织度 | 决定军团能否有效执行移动、进攻和防御 |
| 补给 | 决定组织度恢复、持续作战和非战斗减员 |
| 经验 | 分为新兵、训练有素、老兵三级 |

- 大量补充新兵会按比例稀释经验。
- 合并军团按兵力加权计算经验。
- 和平时期可以支付额外军费和补给进行训练，不减少 POP。

## 将领与统治者领军

每支军团最多任命一名将领。

| 能力 | 作用 |
|---|---|
| 指挥 | 提高有效指挥规模与战斗协同 |
| 机动 | 影响移动、侦察、遭遇和撤退 |
| 围城 | 提高围城进度并降低强攻损失 |

将领可以拥有少量纯军事特质，并可能受伤、战死或被俘。将领不拥有个人声望、野心、职位、封地或家族影响力。

国家领导人和继承人可以亲自统军：

- 其军事能力参与军团结算。
- 影响所在军团士气和本国战争意志。
- 可能受伤、战死或被俘。
- 战死直接触发现有领导人换代机制。
- 被俘会降低战争意志，并可进入和平条款。
- 不产生“威望”等项目中不存在的资源。

## 补给

系统自动从首都、城市、港口和军需仓寻找补给路线。

| 环节 | 规则 |
|---|---|
| 补给源 | 首都、城市、港口、军需仓 |
| 运输路线 | 道路、河流、沿岸海运和受控地块 |
| 运输能力 | 受道路、港口、季节、地形、封锁和敌军控制区影响 |
| 军团消耗 | 受军团规模、炮兵比例、移动和作战影响 |
| 玩家操作 | 设置补给节点、军团优先级和是否就地征粮 |

补给不足先降低组织度，然后产生逃兵、疾病和非战斗减员。战争疲惫只会降低治理、运输和恢复效率，不会凭空制造地块破坏。

## 要塞、围城与占领

| 对象 | 规则 |
|---|---|
| 普通地块 | 军团进入后可建立军事占领 |
| 要塞地块 | 必须通过围城攻克 |
| 要塞控制区 | 阻断敌军深入，保护附近补给路线 |
| 围城方式 | 围困、炮击、挖掘、强攻 |
| 围城进度 | 受城防、驻军、炮兵、补给、将领围城能力和海上封锁影响 |

占领只改变军事控制权，不立即改变法定领土归属。占领军撤离且没有驻军时，原政权可以逐渐恢复控制。

围城、强攻、劫掠和长期占领可能造成建筑损毁、POP 死亡、逃亡和产出下降。

## 撤退与溃退

| 状态 | 规则 |
|---|---|
| 有序撤退 | 玩家主动下令，保留较多组织度 |
| 战败撤退 | 自动退向最近的友方补给节点 |
| 溃退 | 暂时失去控制，可能遭到追击、被俘或解体 |

骑兵擅长追击，但追击过深可能脱离补给。被控制区、海岸或不可通行地形包围时，军团可能无路可退。

## 海军与远洋

舰队逐海域六边格移动，并对所在格与邻近格产生局部制海权。海军负责拦截、封锁、贸易保护、海上补给、运输和登陆。

- 早期舰船只能沿岸航行。
- 科技解锁远洋舰船后，相关新单位才能进入远洋。
- 解锁科技不会让旧舰船自动获得远洋能力。
- 未知远洋地块由迷雾覆盖，具备能力的舰队进入后永久揭开。
- 不设计测绘进度、情报过期、地图交易和地图窃取。

## 战争对地块与国家的影响

```mermaid
flowchart LR
    A["战斗、围城、封锁、占领"] --> B["死亡、伤残、迁移、建筑损毁"]
    B --> C["POP 与地块产出下降"]
    C --> D["税收、兵源与补给能力下降"]
    D --> E["国家战争疲惫上升"]
    E --> F["战争意志下降"]
```

### 战争疲惫

战争疲惫是国家的长期压力，范围为 0 至 100，战后逐步恢复。

主要来源：

- 军事人员死亡和永久伤残。
- 平民死亡与逃亡。
- 被占领人口比例。
- 港口封锁造成的贸易损失。
- 粮食和补给赤字。
- 战争持续时间。

防御战争的合法性、战争目标进展和成功保卫本土可以降低增长速度。地块破坏与经济损失不能重复计算。

### 战争意志

每个参战国独立拥有 0 至 100 的战争意志。

| 区间 | 状态 |
|---:|---|
| 70 至 100 | 坚决作战 |
| 40 至 69 | 开始动摇 |
| 15 至 39 | 寻求和平 |
| 0 至 14 | 接近崩溃 |

战争意志受主战争目标、首都、主力军、战争疲惫、统治者被俘、盟友退出和国内稳定影响。防御与独立战争获得保护，但首都失守、主力崩溃或国内革命仍可使其瓦解。

## 战争分数与贡献

战争分数范围为 -100 至 100，主要来源包括：

- 主战争目标的持续控制。
- 首都和关键要塞。
- 重要战役。
- 敌方主力军崩溃。
- 港口封锁和海上控制。

与战争目标无关的占领只提供少量分数，重复击败同一支残军不能持续刷取战果。

每个参战国另行记录参战贡献，来源包括战斗、占领目标、封锁、伤亡承担和军事援助。贡献决定战后利益，但不改变领土宣称归属。

## 和平谈判

- 战争领袖主持整体和谈。
- 参战国公开要求与贡献决定利益分配。
- 盟友可以单独媾和，但会损失信任并承担外交后果。
- 领土交给拥有宣称和和平要求的国家，不自动交给实际占领者。
- 和约建立停战期，撕毁停战产生严重外交代价。

| 和平要求 | 成本依据 |
|---|---|
| 割让领土 | POP、产出、核心、宣称和战争目标 |
| 赔款 | 金额与支付周期 |
| 解除条约 | 条约重要程度 |
| 建立朝贡、附庸或傀儡 | 目标国规模、战争目标和剩余抵抗力 |
| 承认独立 | 独立战争目标完成度 |
| 释放国家 | 涉及地块和 POP |
| 开放通行或市场 | 条款持续时间 |
| 释放战俘 | 战俘身份和数量 |

过度索取会增加其他国家对胜利者的威胁判断，并降低相关国家关系。

## 系统边界

首版不设计：

- 独立战术战场。
- 工程兵、后勤兵等额外作战兵种。
- 逐批粮草运输。
- 征召训练倒计时。
- 将领个人政治与国家内部家族。
- 复杂军事情报和地图交易。
- 完整国际组织战争法。
