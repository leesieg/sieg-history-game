# 13 - 纪元状态机与信仰纪元实现说明

结论：**11 号方案的转折层全图落地：六纪元状态机贯通（封建→发现→信仰→王权→革命→工业），信仰纪元按阶段三完整实现（印刷→改宗扩散→跨信仰战争→大和会），王权/革命/工业纪元以既有引擎轻量落地（常备军/启蒙扩散/革命传染/蒸汽机/大分流榜）。14/14 测试通过，三条验收（改宗地图随局不同、和会规则可被言明、纪元自然推进）全部命中。至此 11 号方案除阶段四（LLM 战略层）外全部实装。**

## 纪元状态机

| 纪元 | 进入判据（窗口+条件，非硬日期） | 纪元规则（卷轴原文展示） |
|---|---|---|
| 封建 | 开局 | 封建征召与教会权威；黑死病与商路动荡松动基石 |
| 发现 | 君堡陷落 **或** 远洋帆船+全球探索里程碑 ≥2（双触发） | 新航线改写贸易；火药与印刷扩散；采纳进步是政治决策 |
| 信仰 | 宗教改革情势爆发（1455+ 且印刷采纳国 ≥3 且任一大国教会满意 ≤-10） | 改宗地图每局不同；跨信仰战争门槛大降；镇压/宽容/改宗皆有人买单 |
| 王权 | 大和会落幕 | 主权规则约束列国；常备军竞赛；启蒙开始流动 |
| 革命 | 任一国财政压力 ≥75 且思想渗透 ≥60 → 首场革命 | 革命跨国传染；镇压只能延迟 |
| 工业 | 蒸汽机采纳国 ≥3 | 大分流开始；终局纪元 |

- 每次纪元更替弹**纪元卷轴**（转折卡变体）：因果链 + 「新纪元的规则」逐条原文 + 工业纪元附**大分流榜**（前五强排名）
- 顶栏时刻显示当前纪元（`1463 年 · 春 · 发现纪元`）

## 信仰纪元（阶段三主体）

| 环节 | 实现 |
|---|---|
| 活字印刷 | 新进步：意大利三邦（威尼斯/米兰/教皇国）1405 年起知晓，沿边界扩散，1470 年全欧知晓；采纳花 60 金、教会满意 -8（失抄经垄断）、文书效率金钱 +4/季。**印刷是双刃剑**：天主教国家每年教会权威 -1（小册子侵蚀）——宗教改革的前置条件由它自己制造 |
| 宗教改革情势 | 条件触发：火种落在**萨克森**（维滕贝格所在），随即开启信仰纪元卷轴 |
| 改宗扩散引擎 | 每季每地块评分：印刷(2) + 城市市民(1) + 教会不满(0-2) + 邻块已改宗(2)，≥4 改宗（每国每季上限 2 块）——**改宗地图由各国的印刷采纳与教会关系决定，每局不同**（测试实证：局A 法兰西先燃、局B 英格兰先燃） |
| 玩家工具 | 《重申正信》敕令（宗教裁判拉回 2 块，教会 +6 / 动乱 +3 / 平民 -4）；「改宗国教」决议（35% 改宗后可用：全境改宗、没收教产 +80 金、教会 -25、平民 +8、合法性 -10）——亨利八世的选择 |
| 信仰张力仪表 | 第 5 仪表：异端地块占比 ×（教会权力系数），≥45 黄色预警、≥70 红色；大和会后减半 |
| 跨信仰战争 | 信仰纪元中天主教/新教国家互战：无需敌对态度、强度门槛降至 1.4、全局战争容量 5；战争标记"信仰之战"并计数 |
| 大和会 | 信仰战争 ≥2 且 24 季后（或纪元 70 季强制）：跨信仰战争全部停战 28 季，和会立三规——**①教随国定**（改宗内政化：扩散不再跨国界、信仰张力减半、改宗国教不再扣合法性）**②条约神圣**（单方面毁约信任惩罚 ×2）**③常驻使节**（全员外交容量 +1）。规则原文写入卷轴与纪闻因果链，机制逐条落地可测 |
| AI 信仰行为 | 改宗过半且教会弱 → 自行改宗国教；改宗初起且教会强 → 自行镇压——阵营版图由各国结构自决 |
| 信仰地图模式 | 新地图模式「信仰」：天主教金 / 新教蓝 / 东正教紫 / 伊斯兰绿，改宗浪潮可视化 |

## 王权/革命/工业纪元（轻量落地）

| 内容 | 实现 |
|---|---|
| 常备军 | 王权纪元进步：150 金+50 军需，需财政改革或集权 ≥55；军需产出 +20%（季报「常备军体系」），贵族满意 -12——军事革命的财政与政治代价 |
| 启蒙扩散 | 王权纪元起：思想渗透（第 6 仪表）按 印刷×市民权力 增长；与财政压力双红即预警"革命的配方" |
| 革命情势 | 双红触发：平民阶层转为**革命军**（复用叛乱系统+革命标记），邻国思想渗透 +15（**思想不需要护照**）；首场革命开启革命纪元 |
| 蒸汽机 | 革命纪元进步：200 金，需 2 座城市；金钱与军需产出 +30%（季报「工业产能」），商人/行会满意 -10（机器夺业） |
| 工业纪元 | 3 国采纳蒸汽机触发；卷轴附大分流榜——采纳者与旁观者从此不在同一条曲线上 |

| 和会新规则可被言明 | ✅ 三条规则原文在事件链中输出，且机制实测生效（外交容量 +1、毁约惩罚 ×2、张力减半） |

| 全链脚本验证 | ✅ 封建→发现→信仰→王权→革命→工业六段连续推进，每段转折卡含因果链 |
