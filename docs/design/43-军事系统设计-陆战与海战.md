# 第一部分 · 设计总述

## A. 现状的病

| 现状 | 病 |
|---|---|
| 陆战成熟（warfare.js：兵种/将领/士气-组织-补给/地形/占领/围攻/疲惫） | 不是病——报告 33「稳定、覆盖好」。**不重写战斗数学** |
| **海洋维度完全缺失** | combatTypes 只有步/骑/炮；**没有舰队/海战/制海权**；`neighbors` 滤掉海域——**陆军根本过不了海** |
| 38 战列舰 / 37 木材→海军 / 39 远洋帆装 都 unlock「海军」 | **指向一个不存在的系统**（审计 B1 空转） |
| 隔海战争（英↔法、抗奥斯曼、殖民）无法表达 | 没有跨海投送 |

## B. 核心思想

> **军事 = 陆军 + 海军 + 共享层 + 整合。** 海军补上**海洋维度 + 跨海投送**——没有运兵，海军是孤岛；有了运兵，任何隔海战争都先夺制海权。

- 海军照陆军范式平行搭（舰队=军团、海战=`sidePower`、海军将领=将领）。
- **整合**才是「完整」的关键：运兵/两栖、海岸围攻、封锁→经济战、海峡咽喉。

## C. 关键设计决策（讨论结论，已定·全做）

- **C1. 完整海军**：舰队 + 舰种 + 海战 + 制海权 + 封锁 + **私掠** + 运兵/两栖 + 海军将领 + 探索。
- **C2. 舰种与海域按历史**：加莱(地中海/内海)↔风帆(大洋)的真实交互 + 海峡咽喉。
- **C3. 运兵/两栖**：做——陆海整合的命根。
- **C4. 海军能力来源**：**航海科技领域(39) + 港口/沿海 + 木材/海军物资(37)，不设独立海军制度**。
- **C5. 制海权轻量**：控己方港口周边海域 + 封锁=敌舰压在港口海域；不做全图海域控制区。

## D. 接缝 + 回到核心循环

> 物产(木材)/科技(航海)/地理(港口·海峡)→ 舰队 → 海战夺制海权 → 封锁断敌贸易(经济战)/运兵跨海投送(战争维度)→ 反作用回基底与商路 → 循环。

## 2. 海军：舰队与舰种（按历史）

舰队 = 海域地块上的「军团」；单位 = 舰船（舰种 × 数量/强度）。舰种**科技 × 物产 × 地理**三重门（同 A3）：

| 舰种 | 科技(39 航海) | 海况强项 | 物产(37) | 历史角色 |
|---|---|---|---|---|
| 桨帆船/加莱 galley | 1337 基础 | **内海/沿海强；大洋不可** | 木材 | 地中海主力（威尼斯/热那亚/奥斯曼/阿拉贡） |
| 柯克 cog | 1337 基础 | 沿海/北海 | 木材 | 北欧/汉萨 |
| 卡拉克·卡拉维尔 carrack | 远洋帆装(1420) | **大洋强** | 木材+海军物资 | 远洋/探索（葡西） |
| 盖伦 galleon | (~1550) | 大洋/通用 | 木材+海军物资 | 帝国主力 |
| 风帆战列舰 ship-of-line | 战列舰(1650) | 大洋·战列决战 | 木材+海军物资 | 海权巅峰 |
| 护卫舰 frigate | (1650) | 通用·快速 | 木材 | 私掠/护航 |

## 3. 海域与航行

- 海域地块（`isSea`）需**平行的海上邻接/寻路**（现 `neighbors` 只走陆地，要加 sea-neighbors）。
- **海况** = 内海/沿海（地中海·波罗的海·北海·黑海）vs **大洋**（大西洋·远洋）→ 舰种交互的「地形」。
- **海峡咽喉**：直布罗陀（地中海↔大西洋，已特判）、博斯普鲁斯/达达尼尔（↔黑海，君堡）、松德（北海↔波罗的海，丹麦收税）、英吉利海峡。**控海峡 = 控通行 + 收过路费**（接 37）。

## 4. 海战解算（navalPower，类 sidePower）

```
navalPower = Σ 舰船强度 × 舰种海况系数(海况) × (1+经验) × 海军将领指挥 × 士气/组织/补给
// 海况系数：加莱 内海×1.2/大洋×0；柯克 沿海×1/大洋×0.6；卡拉克+ 沿海×1/大洋×1.2
// 火炮纪元后：装炮风帆 > 加莱（gunnery 系数）——勒班陀后桨帆退场的机制化
```
- 复用陆战的士气-组织-补给-经验-将领框架，只换「地形」为「海况」、「将领」为「海军将领」。

## 5. 制海权 + 封锁 + 私掠

- **制海权（轻量, C5）**：舰队控制己方港口/商路周边海域；敌舰进入则争夺。
- **封锁 blockade**：舰队压在敌港海域 → **切其经由该港的商路**（接 37 市场）、抬其财政压力、堵其舰队出港。
- **私掠 privateering**：轻量劫掠动作——舰队袭敌商路（非全封锁）→ 窃取贸易收入、低投入、伤声誉（38）。经济战的小刀。

## 6. 运兵 / 两栖（陆海整合命根）

- 舰队**载运陆军**跨海；在沿海地块**两栖登陆**。
- **需航线制海权**：敌舰可**拦截**运输舰队（运兵途中被截 = 军团损失/落海）。
- → **隔海战争先夺制海权**：英法、抗奥斯曼、殖民远征全靠它。

## 7. 海军将领 + 探索 / 殖民

- **海军将领 admiral**：复用将领系统（指挥/经验），来源同 C4（航海研究/港口/海事阶层）。
- **探索/殖民**：远洋舰队探索大洋 → 发现新大陆 → 殖民物产/跨洋商路（接 37 命名大商路 + 39 航海树）。

## 8. 整合 陆↔海（这才叫「完整」）

| 整合点 | 机制 |
|---|---|
| 跨海战争 | 陆军隔海作战 = 运输舰队 + 航线制海权（解锁 38 跨海战争目标） |
| 海岸围攻 | 舰队封锁/炮击支援沿海城市围攻 → 加速占领 |
| 封锁→经济战 | 制海权→封港/私掠→断敌贸易流（37），不流血扼杀 |
| 海峡咽喉 | 控海峡 = 控通行 + 过路费 + 咽喉海战 |
| 共享层 | 将领/海军将领、士气-组织-补给、战争分数、三重门 gating——陆海共用 |

## 9. 接缝（与审计 A3 一致）

| 接缝 | 机制 |
|---|---|
| 海军能力来源 | **航海科技领域(39) + 港口/沿海地理 + 木材/海军物资(37)**；**无独立海军制度**（maritime 槽已废，C4） |
| 舰种 gating | 科技 × 物产 × 地理（§2） |
| 制海权 → 经济 | 封锁/私掠断商路、海峡过路费（37） |
| 运兵 → 战争 | 解锁跨海战争目标（38） |
| 海事阶层 | port_nobles/sailors/companies（商业共和国）→ 海军能力/海军将领 |

## 12. 风险

| 风险 | 缓解 |
|---|---|
| 海军把军事复杂度翻倍 | 严格复用陆战框架（士气/组织/补给/将领/战争分数），只换地形→海况 |
| 海域寻路/AI 性能 | 海况粗粒度、制海权轻量（C5）；带默认 getter；invariants 守门 |
| 内陆国被海军边缘化 | 海军只对沿海/隔海战争关键；内陆战争纯陆战，不受影响 |
| 加莱/风帆平衡 | 海况系数 + 火炮纪元 gunnery 转折，按史实标定，sim 调 |
| 运兵拦截过狠/过弱 | 拦截概率随制海权差缩放，sim 调 |
