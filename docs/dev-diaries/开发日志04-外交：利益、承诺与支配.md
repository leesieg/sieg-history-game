# 开发日志 04 - 外交：利益、承诺与支配

大家好，今天聊外交。

在《帝国的代价》里，外交不是一条从红色刷到绿色的好感度，也不是把金钱送给邻国，然后等待“结盟”按钮亮起。

国家会合作，是因为合作暂时符合利益；国家会履行承诺，是因为背叛存在代价；国家会屈服，则可能是因为它已经没有更好的选择。

我们的目标很直接：

**让外交关系同时表达彼此怎么看、承诺了什么，以及谁有权命令谁。**

## 1337 年没有平等的国际棋盘

1337 年的欧洲、北非和地中海世界，并不是一群边界清晰、主权完全平等的现代国家。

英格兰国王同时拥有法国境内的领地，并对法国王位提出主张；法兰西王权试图把封建关系转化为更直接的国家统治；神圣罗马帝国的诸侯既是独立行动者，也是帝国政治体系的成员；拜占庭依靠外交、婚姻和权宜联盟维持残存秩序；意大利城邦会在商业合作、领土竞争和帝国派系之间反复转向。

北非和安纳托利亚同样存在贡赋、保护、附庸、边疆自治和名义臣服等不同关系。

| 时代现象 | 游戏中的问题 |
|---|---|
| 王室联姻连接多个政权 | 亲属关系是否会改变联盟与继承 |
| 封臣拥有自己的领地和军队 | 臣服是否等于忠诚 |
| 商业城市依赖跨国市场 | 敌对国家是否仍可能保持贸易 |
| 帝国成员共享政治秩序 | 组织成员是否等于皇帝附庸 |
| 弱国以贡赋换取生存 | 主权损失能否换来安全 |
| 君主个人关系影响国策 | 领导人换代后承诺是否继续有效 |

如果外交只有“友好”和“敌对”，这些结构都会被压成同一种东西。

## 国家态度不是正式关系

外交系统的第一层是国家态度。

态度不是玩家直接选择的标签，而是国家根据地图、历史和当前局势自动形成的判断。

| 态度 | 常见原因 | 行为倾向 |
|---|---|---|
| 亲近 | 长期履约、低威胁、利益一致 | 联姻、同盟、共同防御 |
| 合作 | 存在贸易或共同敌人 | 协定、通行、有限援助 |
| 中立 | 缺少直接利益与冲突 | 观察局势、接受有限往来 |
| 警惕 | 对方扩张或边境军力上升 | 寻找保证、建立制衡 |
| 竞争 | 争夺领土、市场或地区影响力 | 拉拢盟友、阻碍对方 |
| 敌对 | 高威胁、高矛盾、低信任 | 最后通牒、战争准备 |

这些态度来自六类方向性因素：

- 信任。
- 威胁。
- 恩义。
- 领土矛盾。
- 制度冲突。
- 战略利益。

“方向性”很重要。

一个小国可能把强邻视为巨大威胁，而强国只把这个小国视为无关紧要的边境问题。附庸可能敌视宗主，宗主却认为双方关系稳定。两国共享同一份条约，但不需要共享同一种感受。

正式契约也不会覆盖国家态度。

盟友可能因共同敌人暂时合作，同时仍在争夺同一片土地；贸易伙伴可能互相信任，却因宗教和继承问题拒绝联姻；帝国诸侯可以同属一个组织，也可以在帝国规则允许的范围内互相竞争。

## 外交的四层结构

为了表达这些差异，我们把外交关系拆成四层。

| 层级 | 回答的问题 | 例子 |
|---|---|---|
| 国家态度 | 双方目前如何看待彼此 | 合作、警惕、竞争、敌对 |
| 外交契约 | 双方正式承诺了什么 | 贸易、通行、联姻、互不侵犯、同盟 |
| 权利结构 | 一方能否支配另一方 | 朝贡、附庸、傀儡、共主、边疆属国 |
| 国际组织 | 双方是否受共同规则约束 | 帝国诸侯、选帝侯、自由市、宗教联盟 |

这四层可以同时存在。

一个国家可以敌视自己的宗主，与邻国保持贸易，又同宗主一起参加防御战争。两个帝国诸侯可以签订联姻，同时围绕边境继承权展开竞争。

外交不再要求每对国家只能拥有一个标签，而是让不同关系在同一时间共同发挥作用。

## 玩家如何改变关系

玩家不能直接点击“提高友好度”。

你能做的是派遣使节、支付代价、提出契约、作出承诺，或者施加压力。国家态度会根据这些事实逐步改变。

### 外交资源

| 资源 | 作用 | 限制 |
|---|---|---|
| 外交行动点 | 发起提案、赠礼、会晤和威慑 | 由领导人外交能力按季产生 |
| 使节 | 执行改善关系、安抚附属国等持续任务 | 同一时间只能执行有限任务 |
| 外交容量 | 维持同盟、保证与附属关系 | 超额承诺会降低外交效率和信任 |

外交点代表国家抓住机会的能力，使节代表持续投入，外交容量则代表国家能否长期维护自己的承诺。

一个外交能力优秀的领导人可以更频繁地进行会晤和谈判，但无法凭空维持十几个同盟。国家可以继续超出外交容量签约，却会让盟友怀疑承诺是否可信，也会让附属国感到宗主已经无力管理整个体系。

### 使节任务

使节行动不会立刻改变国际局势，而是按季度推进。

改善关系会逐步提高信任、降低威胁；安抚附属国可以提高忠诚，但会占用本可用于其他方向的使节。

这意味着外交部署也是一种资源选择。

当英格兰与法兰西的冲突不断升级时，你可以派使节稳住一个潜在盟友，也可以安抚一个不满的附庸，却很难同时照顾所有方向。

### 即时行动

| 行动 | 直接效果 | 代价或风险 |
|---|---|---|
| 赠礼 | 提高信任和领导人友谊 | 消耗金钱与外交点 |
| 私人会晤 | 提高友谊与尊重 | 效果可能随领导人换代衰减 |
| 公开威慑 | 提高恐惧和短期屈服意愿 | 增加威胁与宿怨 |
| 贸易协定 | 增强共同经济利益 | 可能强化国内商人阶层 |
| 军事通行 | 开放军队路线 | 承担外国军队接近本土的风险 |
| 王室联姻 | 建立亲属和继承联系 | 可能卷入新的王位争端 |
| 互不侵犯 | 降低短期战争风险 | 限制自己的扩张空间 |
| 防御同盟 | 获得外部保护 | 必须承担参战义务 |

每项外交行动都应该改变国家所处的约束，而不是只提供一个临时修正值。

## 提案不是掷骰子

外交提案采用确定性判断。

在提交前，玩家可以看到对方为什么接受，也能看到为什么拒绝。

```text
接受意愿 =
对方直接收益
+ 战略利益
+ 国家关系
+ 领导人关系
+ 共同威胁
+ 国内支持
- 条约成本
- 主权损失
- 对提案方的威胁
- 现有契约冲突
```

| 结果 | 含义 |
|---|---|
| 接受 | 条件达到门槛，契约生效 |
| 拒绝 | 对方认为收益不足或风险过高 |
| 可谈判 | 条件接近，对方可能要求金钱或让步 |
| 不可提交 | 提案与战争、停战、制度或现有契约直接冲突 |

我们不希望外交成为反复点击、等待随机成功的过程。

真正的选择应该发生在提交之前：你愿不愿意支付更多代价，是否应该先削弱威胁，或者是否需要等共同敌人出现后再提出同盟。

## 领导人能够打开窗口，但不能代替国家

国家关系与领导人私人关系分别保存。

腓力六世与爱德华三世之间的王位矛盾，不等于两国所有利益的总和；威尼斯总督与外国君主建立的尊重，也不会自动消除港口和市场竞争。

| 私人关系 | 外交作用 |
|---|---|
| 友谊 | 提高合作提案接受意愿 |
| 亲属 | 支持联姻与继承外交 |
| 尊重 | 提高会晤和调停效果 |
| 恐惧 | 提高屈服型提案的短期效果 |
| 宿怨 | 降低合作意愿，提高冲突升级风险 |

领导人换代时，友谊、尊重、恐惧和宿怨会逐步衰减，私人承诺可能直接失效，但正式国家契约仍然存在。

一位新国王可以重新评估旧联盟，却不能假装旧联盟从未签署。个人外交制造机会，国家制度决定机会能否变成长期秩序。

## 臣服不等于忠诚

外交系统还需要表达不平等关系。

我们不会把朝贡国、附庸国和傀儡国统一成一个“属国”按钮。

| 类型 | 核心关系 | 常见特点 |
|---|---|---|
| 朝贡国 | 以贡赋换取承认或保护 | 外交自主较高，军事义务较低 |
| 附庸国 | 承认宗主权并承担契约义务 | 保留部分外交和内政权 |
| 傀儡国 | 对外政策受宗主控制 | 自主权低，通常强制参战 |
| 共主邦联 | 多国共享同一领导人 | 内政独立，外交部分绑定 |
| 边疆属国 | 以防务换取自治 | 军事义务高，贡赋较低 |

每个附属关系分别记录两个关键状态：

| 状态 | 含义 |
|---|---|
| 法定自主权 | 契约允许附属国自行决定多少事务 |
| 现实忠诚度 | 附属国是否愿意履行这些条款 |

低自主权不代表高忠诚。

宗主可以提高贡赋、限制外交、干预继承并强制附属国参战，但这些权力会降低忠诚。忠诚过低后，附属国会拒绝贡赋、寻找外部支持、要求提高自治，最终可能发动独立战争。

反过来，一个拥有较高自主权的边疆属国，也可能因为宗主长期提供保护而保持忠诚。

玩家面对的不是“怎样把自主度压到最低”，而是“怎样让这个关系仍然值得双方维持”。

## 国际组织不是大型附庸系统

神圣罗马帝国这样的政治体系不能用普通宗主关系表现。

帝国诸侯拥有自己的领导人、领土、外交和军队。皇帝拥有保护、调停和推动改革的权力，但不能把每位诸侯都当成可以直接命令的附庸。

| 组织身份 | 特殊权利与义务 |
|---|---|
| 普通诸侯 | 保留内政，在帝国法下承担防务 |
| 选帝侯 | 参与皇帝选举 |
| 自由市 | 获得保护和贸易特权 |
| 皇帝 | 保护成员、调停争端、推动改革 |

当前版本已经保留国际组织的成员与身份结构，但完整的选举、议案、改革、组织执法和成员战争规则会在双边外交与战争系统稳定后开发。

多边制度必须建立在可靠的国家关系和条约义务之上，否则只会变成另一组脱离世界的按钮。

## 外交如何影响国家内部

外交并不只发生在两个国家之间。

每项对外决定都会在国内制造受益者和反对者。

| 外交行为 | 可能受益者 | 可能反对者 |
|---|---|---|
| 贸易协定 | 商人、行会、港口城市 | 受竞争冲击的本地生产者 |
| 王室联姻 | 宫廷、贵族、王室官僚 | 敌对家族、宗教集团 |
| 主动战争同盟 | 军事贵族、边疆集团 | 平民、商人、议会反战派 |
| 向强国屈服 | 希望维持和平的阶层 | 贵族、军队和集权派 |
| 提高附属贡赋 | 宗主官僚和财政集团 | 附属国阶层 |
| 放宽附属自治 | 附属国地方集团 | 宗主中央集权派 |

政体也会改变外交授权方式。

君主可以依靠王权快速签订部分契约，但可能激怒等级会议；共和国需要议会支持，却能让长期承诺更稳定；商业共和国更容易推动贸易协定，但主动战争可能遭到商人反对；帝国需要考虑诸侯、组织法律和共同防务义务。

外交因此成为国内政治的延伸。

## 从外交走向战争

外交不是战争之外的和平小游戏。

它决定战争为什么发生、谁会参战，以及战争结束后形成什么秩序。

```mermaid
flowchart LR
    A["国家利益与态度"] --> B["使节、提案与施压"]
    B --> C["契约或从属关系"]
    C --> D["履约、违约与冲突"]
    D --> E["战争目标与参战义务"]
    E --> F["和平条款与新秩序"]
    F --> A
```

同盟会产生参战义务，互不侵犯会限制宣战，军事通行会改变军团路线，从属条款会决定附属国是否必须出兵。违约会降低信任，并可能为对方提供战争理由。

战争胜利也不只是改变边界。和平条约可以要求赔款、领土、贡赋、附庸化、独立或解除契约，从而重新组织外交体系。

## 当前原型

外交模块已经进入可操作原型。

| 特性 | 状态 |
|---|---|
| 双边方向性关系 | 已完成原型 |
| 国家态度自动生成 | 已完成原型 |
| 外交点、使节和外交容量 | 已完成原型 |
| 改善关系与安抚附属国 | 已完成原型 |
| 赠礼、私人会晤和公开威慑 | 已完成原型 |
| 确定性提案接受意愿 | 已完成原型 |
| 贸易、通行、联姻、互不侵犯、防御同盟 | 已完成原型 |
| 契约期限、退出、违约与季度履约 | 已完成原型 |
| 朝贡、附庸、傀儡、自主权与忠诚度 | 已完成原型 |
| 附属国拒贡与独立战争 | 已完成基础原型 |
| 国际组织成员与身份展示 | 已完成底层结构 |
| 战争分数、基础和平与停战 | 已接入战争原型 |
| 组织选举、议案、改革与执法 | 后续开发 |
| 完整历史外交数据库 | 持续校准 |

当前原型的重点不是提供最多的外交按钮，而是验证关系能否被玩家理解。

当一个国家拒绝同盟时，玩家应该能看见它担心什么；当附属国拒绝贡赋时，玩家应该知道是战争失败、贡赋压力还是宗主干预造成的；当一份契约失效时，世界应该记住谁违背了承诺。

## 本篇结论

外交系统的设计原则是：

**态度来自利益，合作依靠承诺，支配必须支付维持秩序的代价。**

下一篇开发日志会讲战争与和平：军团如何在地图上移动、战斗与围城，以及一场战争怎样通过伤亡、占领和破坏真正改变国家。
